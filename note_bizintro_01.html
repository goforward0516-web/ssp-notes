📋 講義概要・学習目標

本科目「企業経営入門」は、企業経営に関わる基礎的な概念と仕組みを体系的に学ぶ15回講義である。第1回はオリエンテーションとして、講義全体の目標・構成を把握し、企業とは何か・ビジネスモデルとは何かを大まかに理解することを目的とする。 ← S03:p.2

  • この講義の目標・方針・概要を理解する
  • 企業経営とは何かをおおまかに把握する

第1章 イントロダクション

1-1 この講義の達成目標

以下の項目を理解し、自分の言葉で説明できるようになることが目標。単なる暗記ではなく、自分なりの理解を求めている。 ← S03:p.4

「自分の言葉で説明できるようになってほしい」という表現には、内容を暗記するのではなく、しっかりと理解してほしいという教員の意図が込められている。 ← S05-A
  • 株式会社の仕組みと企業経営を構成する要素
  • 経営理念・経営戦略・マーケティング・技術経営の概念とそれらの関係
  • 企業の資金調達、財務諸表の種類と役割
  • 管理会計とその主要分析手法

これらは、将来企業に就職する人にとっても役立つ内容として設計されている。経営者としての視点だけでなく、従業員として知っておくべき知識も含まれている。 ← S05-A

1-2 なぜ企業経営を学ぶのか

企業経営を学ぶ理由は以下の通り。 ← S03:p.6

  • 企業は経済活動の中心である
  • 多くの人が企業で働いている（就業者の約88%が雇用者、約12%が自営業主・家族従業者）
  • 経済主体は「家計・企業・政府」の3つであり、企業が大きな位置づけを占める
  • モノやサービスを生み出す主体の多くは企業である
  • 経営系科目の位置づけを把握し、関連科目への関心を持つ
就業者の約88%が雇用者という数値は「労働力調査 2017年5月分」に基づく（スライド掲載値）。受講生の多くが企業と関わりを持っているという文脈で提示された。 ← S03:p.6, S05-A
[外部補足] 最新の総務省「労働力調査（基本集計）2025年平均」では、就業者6,842万人のうち雇用者が6,227万人（約91%）、自営業主・家族従業者が589万人（約9%）。雇用者比率は2017年以降も上昇傾向が続いている。（出典: 総務省統計局 stat.go.jp）

1-3 家計・企業・政府の関係

企業は家計（消費者）に財・サービスを提供し、代金を受け取る。また、家計から労働力の提供を受け、賃金を支払う。政府は家計・企業から税金を受け取り、道路・水道・治安維持・防衛などの公共サービスを提供する。 ← S05-A

1-4 企業と会社の違い

用語概念法律上の定義
会社 私益を目的とする営利社団法人。利益追求を目的とした法人格を持つ組織。 あり（会社法で規定）
企業 広く「事業を営む経済単位」という概念。会社より大きな括り。 なし

← S03:p.8

日常会話では「会社」と「企業」は混用されることが多いが、「会社」は法律上の位置づけがある。教員も時々混用することがあるため、注意して使い分けてほしいと説明している。 ← S05-A

1-5 企業の種類・会社の種類

一般的に「会社」と言えば株式会社を指すことが多い。公企業（国・地方公共団体の企業、公団・公社）についてはこの講義では詳しく扱わない。 ← S05-A

会社の種類の比較

株式会社合同会社合名会社合資会社NPO（参考）
責任形態 有限責任 有限責任 無限責任 有限＋無限 なし
社員の入退社 買受・譲渡可能 総社員の同意 総社員の同意 総社員の同意 総社員の同意
納税 あり あり あり あり なし
配当 あり 自由 あり あり なし

← S03:p.10

【定義】有限責任：責任が出資の範囲に限られること。無限責任：出資の範囲を超えて個人の財産まで責任が及ぶこと。
⚠️ 表中の「社員」とは法律上の概念であり、株式会社でいう「株主」にあたる（一般的な「社員＝従業員」とは異なる）。 ← S03:p.10
株式会社の2つの重要な特徴：①責任が有限であること（出資した分しか責任を取らなくてよい）、②株式の買受・譲渡が可能であること（他の会社形態は総社員の同意が必要）。この2点から、多くの人が関わりを持てる形態となっている。 ← S05-A

1-6 株式会社の歴史

当時のアジアとの香辛料（胡椒）貿易はハイリスク・ハイリターンであった。船を出して貿易を行い、利益を出資者に還元する「当座企業」（航海ごとに資金を集める一時的な企業）として始まった。 ← S05-A, S03:p.11
[外部補足] オランダ東インド会社の正式名称はオランダ語で「Vereenigde Oost-Indische Compagnie（略称: VOC）」。資本金は約650万ギルダー、重役会は「17人会（Heeren XVII）」と呼ばれた。本社はアムステルダム。アムステルダム証券取引所と同時に設立され、1799年12月31日にオランダ政府により解散。学術的には「①全社員の有限責任制、②会社機関の存在、③譲渡自由な等額株式制、④確定資本金制と永続性」という4指標を初めて具備した組織として、株式会社の起源とされている（中野常男の分析による）。（出典: Wikipedia「オランダ東インド会社」ほか）
  • 1600年イギリス東インド会社（EIC）設立 ── 航海ごとに出資者を集める方式（当座企業）で始まった。VOCの前身にあたる形態。
  • 1602年オランダ東インド会社（VOC / Vereenigde Oost-Indische Compagnie）設立 ── 世界初の株式会社とされる。EICと異なり最初から株式を発行して多額の資本を集めた点が画期的。資本金約650万ギルダー。
  • 1657年イギリス東インド会社が株式会社化 ── 継続的な株式会社の最初とされる。4つの特徴：①全出資者が有限責任、②取締役会の設置、③資本の証券化、④永続企業。
  • 18世紀初株式会社ブームと泡沫会社（bubble company）の乱立
  • 1720年泡沫会社禁止法制定（イギリス） ── バブル崩壊・「南海の泡沫事件（South Sea Bubble）」
  • 18世紀後半産業革命 ── 多様な産業で大規模資本が必要に
  • 1799年オランダ東インド会社（VOC）解散 ── オランダ政府により解散命令。約200年間の歴史に幕。
  • 1825年泡沫会社禁止法の廃止（イギリス）
  • 1844年特許制度から登記制度へ移行（イギリス） ── 国の許可なく登記だけで株式会社設立が可能に
  • 1855年有限責任法（イギリス）
  • 1862年会社法（イギリス） ── 現代に近い形で成立

← S03:p.11–13, S05-A

[外部補足] 「南海の泡沫事件（South Sea Bubble）」は1720年のイギリスで起きた投機ブームと崩壊。南海会社の株価が急騰・暴落し、多くの投資家が損失を被った。これが泡沫会社禁止法（Bubble Act 1720）制定のきっかけとなった。 [外部補足]

第2章 企業経営とは何か

2-1 企業の目的

【定義・重要】「企業の目的は顧客を創造することである。利益は企業の目的ではなく、存続の条件である。」（ピーター・F・ドラッカー） ← S03:p.16
企業は利益がなければ継続できない（going concern＝継続的事業活動）。しかし、その利益は財・サービスを買ってくれる顧客なしには生まれない。短期的な利益だけを追うのではなく、顧客を創造し続けることが企業が存続するための本質的な目的だという考え方がドラッカーの主張。この「利益は目的か手段か」という問いが、この講義全体の大きなテーマのひとつである。 ← S05-A
[外部補足] ピーター・F・ドラッカー（Peter F. Drucker, 1909–2005）はオーストリア生まれのアメリカの経営学者。「現代経営学の父」とも呼ばれ、マネジメントの概念を体系化した。代表著作に『マネジメント』（1973年）がある。 [外部補足]

この講義では「企業」は原則として「私企業（営利企業）」を指す。 ← S03:p.16注

2-2 財とサービスの分類

企業が生み出すものは「財」または「サービス」である。 ← S03:p.17

財の分類

分類軸種類A種類B備考
用途 消費財（買って使うもの） 生産財（企業が生産に使うもの。機械・設備等） 生産財は減価償却の対象
形態 有形財（形のある財） 無形財（知的所有権・情報財） 特許・商標権・デジタルデータ等
耐久性 耐久消費財（車など長期使用） 非耐久消費財（食品など即時消費） 中間に半耐久消費財
耐久消費財・非耐久消費財の区分けは税務・会計制度においても重要な概念。情報財はデジタルなやり取りが可能な形のない情報を扱うものを指す。 ← S05-A

サービスの特性

特性説明例
同時性 生産と消費が同時に行われる。顧客がいないと提供できない。 理髪店・レストランでの提供
消滅性 在庫として蓄積できない。 飛行機の空席・旅館の空室は翌日に繰り越せない

← S03:p.17

2-3 企業活動の循環

企業は自社の資源をどう配分するかを決め、競争相手より優位に立てるよう事業を運営する。そこで生まれた付加価値を顧客に提供することで利益を得る。この利益をさらに再投資することで、企業活動が継続・成長していく。 ← S05-A

2-4 市場と競争

【定義】市場（マーケット）：経営の世界では「買い手の集合」を指す（原義は買い手と売り手が集まる取引の場）。 ← S03:p.19
概念説明
市場競争 買い手を獲得するために、価格・品質などの面で他の企業と競い合うこと
差別化・差異化 独自の付加価値を付けることで単純な価格競争から抜け出す戦略
ブルーオーシャン 競争のない未開拓市場
レッドオーシャン 競争が激しい市場（「血に染まった」赤い海のイメージ）

← S03:p.19

「市場分析」とは、自社の製品・サービスを買ってくれる人がどこにいるかを分析すること。企業が継続的に事業を続けるためには、自分の顧客をどこに求めるかという観点が非常に重要。 ← S05-A

2-5 持続的な優位性の構築

【定義】競争優位：業界平均以上の利益率、または一定以上の市場シェアを持つ状態。コスト優位と差別化優位（差異化優位）がある。 ← S03:p.20

模倣困難性（競争相手が簡単に真似できない「何か」）

要素説明
特許・著作権法律で保護され、他者が同じものを作れない
ブランド長く市場で認知されることで競合が追随しにくい
立地人通りの多い場所・一等地での商売は他者が真似できない
許認可国や行政から取得した特定業種の参入許可

← S03:p.20

持続的な競争優位が大事な理由：長期的に競争優位な企業こそが事業を継続できるから。目の前の利益だけを追う企業は長続きしない。 ← S05-A

2-6 ステークホルダーとCSR

【定義】ステークホルダー（Stakeholder）：企業と利害関係を持つ個人・法人・機関。株主・従業員・労働組合・仕入先・顧客・外注先・政府・地域住民・コミュニティなど。 ← S03:p.21
ステークホルダーの協力・関与なしに、企業は継続的に付加価値創造活動を続けられない。たとえば社員が全員辞めてしまったり、仕入先が商品を納入してくれなくなれば、企業は運営できなくなる。ステークホルダーとの関係をいかに良好に維持するかが経営の重要ポイント。 ← S05-A
【定義】CSR（Corporate Social Responsibility：企業の社会的責任）：社会的な公正さ（コンプライアンス）・社会貢献・地域貢献・環境への配慮・従業員対応など、企業が公器として果たすべき責任。 ← S03:p.21
[外部補足] コンプライアンス（compliance）は「法令遵守」を中心としつつ、それにとどまらず企業が守るべき行動規範全般を含む概念。近年は「法令遵守」だけでなく「社会規範・倫理的行動」まで含む広義の使われ方が一般的。 [外部補足]

2-7 リスクマネジメント

企業を取り巻くリスクは多様化している。 ← S03:p.22

  • 自然災害（大地震・台風・洪水等）、異常気象
  • 戦争・テロ・火災・誘拐
  • 社会インフラ（電気・ガス・鉄道等）の停止
  • 金融危機・原油高騰・金利変動・為替変動
  • 不正取引・談合・横領・輸送事故
  • 不正アクセス・情報漏えい・情報システム停止
  • 投資の失敗・製品事故・リコール など
【定義】ERM（Enterprise Risk Management：エンタープライズリスクマネジメント）：企業全体のリスクを統合的に管理する手法・考え方。 ← S03:p.22
多様なリスクに対して常日頃から備えておくことが企業経営に求められている。コンプライアンスと企業倫理も、このリスクマネジメントの観点から厳しく求められる。 ← S05-A

2-8 企業経営を構成する要素

この講義全体で学ぶ企業経営の構成要素（専門家によって分け方は異なる）: ← S03:p.23

これがこの講義の15回を通じて学ぶ内容の全体像。経営理念・企業の価値観が最上位にあり、その下に各種マネジメント要素が連携している。 ← S05-A

第3章 ビジネスモデル

3-1 ビジネスモデルとは何か

【定義】ビジネスモデル：企業が収益を得る仕組み。顧客や他の企業等との間で財・サービス・情報をどういう経路でやり取りし、その中でどのように収入を得て利益を上げていくかという仕組み。 ← S03:p.26
ビジネスモデルという用語はよく聞かれるが、具体的に説明した書籍はあまりない。3つの視点（戦略モデル・オペレーションモデル・収益モデル）から説明できるが、どのモデルにも「①顧客に何を提供するか、②どのように届けるか、③どれだけ儲かるか」という3要素が必ず含まれていなければならない。 ← S05-A
モデル説明
戦略モデル顧客に提供するものは何か
オペレーションモデル事業活動の構造と前提
収益モデル事業活動の対価を得る構造と前提

← S03:p.26

3-2 各業態のビジネスモデル例

パン屋・製造業・流通業

いずれも基本構造は同じ：売上総利益（粗利益）＝ 売上高 ― 売上原価（商品の原価） ← S03:p.27–29

業態仕入生産販売売上原価に含まれるもの
パン屋 小麦粉 パンを焼く 店頭・無店舗 小麦粉代・焼釜設備・店舗費
製造業 原材料・部品 組み立て 販売店経由 原材料費・労務費・光熱水料・減価償却費
流通業（小売） 商品 （なし） 店頭 商品仕入れ費・労務費・光熱水費・減価償却費
「安く仕入れて高く売る」が商売の原則。個別商品の価格設定において、仕入れと売値の差（価格差）がこれらのコストを賄えるよう設定されなければならない。 ← S05-A

新聞社のビジネスモデル

収益源が2つある点が特徴。 ← S03:p.30

売上総利益 ＝ 購読料 ＋ 広告料 ― 売上原価

専門雑誌も同様に購読料収入と広告料収入の2本立て。フリーペーパーは購読料がゼロであるため、収入はほぼ広告料のみ。そのため広告スペースの割合が大きくなっている。 ← S05-A

3-3 情報仲介型ビジネスモデル

顧客紹介型（例: 中古車情報サイト）

買い手と売り手をつなぐ情報仲介業。収入源：カーディーラー等からの紹介料（または広告料）。インターネット普及後に急増したモデル。 ← S03:p.31

情報サイトが多数出てきたことで紹介料を取りにくくなり、単純にメーカーから広告料を取るモデルに移行しているサイトも増えている。 ← S05-A

オークション型（例: オークションサイト）

売り手と買い手をマッチングし、複数の買い手が入札して売り手と契約する仕組み。収入源：出品料・落札手数料。近年は手数料・出品料無料のサイトも増加。 ← S03:p.32

ビジネスモデルの表現方法

「物の流れ」と「代金の流れ（逆方向）」を図で整理することが有効。重要なチェックポイント：各取引関係で代金がしっかり回収できるか（例：部品代金が高騰した場合に販売店に転嫁できるか）。 ← S03:p.33

新しいサービスやベンチャー企業が上場するとき、「どんなビジネスモデルで儲けているのか」を図にして整理してみることを勧めている。自分の事業立ち上げ時にも有効な思考整理ツール。 ← S05-A
