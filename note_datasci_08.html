第1章

正規分布

学習目標：正規分布の特徴や性質について説明できる S03:p.7

ヒストグラムと確率分布
🎙 AUDIO — ヒストグラムの復習（第3回）

ヒストグラムは度数や相対度数をビン（bin）の高さで表現したグラフ。ビンの横幅×高さ＝面積に注目するのが重要。また、平均・メジアン・モードの位置関係で分布の形（左右対称 / 右偏 / 左偏）が読み取れる。← S05 / S03:p.13

相対度数のヒストグラムにおいて、全ビンの面積の合計は必ず 1 になる（累積相対度数が1になることに対応）。S03:p.14

ビンの横幅をどんどん細かくし、データ数を増やすと、ヒストグラムは滑らかな曲線（確率分布）に近づく。この曲線で囲まれた面積＝確率という関係が成立する。S03:p.14

EXAMPLE — お茶碗のお米の重量

適当にお茶碗にお米をよそった場合の重量に確率分布があるとする。140g〜160gで囲まれたエリアの面積が 0.55 であれば、「次に盛った場合も55%の確率で140〜160gの範囲に入る」と言える。← S05 / S03:p.15

正規分布とは
DEFINITION
正規分布（Normal Distribution）

統計学で最も重要な確率分布の一種。形状の特徴は以下の通り。S03:p.16-17

  • 左右対称の釣鐘型（ベル型）
  • 平均値 = 中央値 = 最頻値（一致）
  • 分散（σ²）と平均（μ）によって形が完全に決まる（最重要）
🎙 AUDIO — 正規分布の式について

正規分布の確率密度関数は授業では暗記不要。ただし、式の中に分散 σ² と平均 μ がしっかり含まれている、という点だけ確認しておくこと。また、以降のスライドでは標準偏差 √σ² を σ（シグマ）と略記している（標準偏差は分散の正の平方根）。← S05 / S03:p.17

σ（標準偏差）が大きくなるほど分布は平べったく（横に広く）なる。これは標準偏差がデータのばらつきを表す指標であることに対応する。S03:p.18-20

図：σの違いによる正規分布の形状変化 ← S03:p.18-21
DEFINITION
標準正規分布（Standard Normal Distribution）

σ = 1、μ = 0 の正規分布を特別に標準正規分布と呼ぶ。統計学で特によく登場する。S03:p.21

正規分布の特徴①：σによって確率が決まる

正規分布では、平均 μ と標準偏差 σ が分かれば、任意の範囲に含まれるデータの割合（確率）が計算できる。S03:p.22-27

図：正規分布の面積（σ範囲と確率の関係）← S03:p.22-26
範囲 データの割合 ポイント
μ ± σ 68.3% 約2/3のデータが含まれる
μ ± 2σ 95.4% ほぼ全てのデータが含まれる
μ ± 3σ 99.7% ほぼ確実に含まれる
μ ± 1.96σ 95.0%（ちょうど） ★検定で頻出する数値
KEY NUMBER — 覚えておく数値
1.96σ = 95% エリアの境界

2σ（≈95.4%）よりも少しだけ中心寄りの値。第3章の検定でも繰り返し登場する。← S05 / S03:p.27

正規分布の特徴②：自然界・人間の特性によく当てはまる

正規分布が統計学で重要な理由は、人間の行動・身体特性・自然現象が正規分布に従うことが多いためである。S03:p.28

EXAMPLE — 身長の分布

20歳以上の日本人男性の身長：平均 167.7cm、標準偏差 6.7cm（厚生労働省「令和元年 国民健康・栄養調査」）S03:p.29

  • μ ± σ = 167.7 ± 6.7 cm → 68.3% の男性が含まれる
  • μ ± 2σ = 167.7 ± 13.4 cm → 95.4%
  • μ ± 3σ = 167.7 ± 20.1 cm → 99.7%
  • μ ± 1.96σ = 167.7 ± 13.1 cm → 95%（154.6〜180.8 cm）
🎙 AUDIO — 実用的な意義

この知識があると例えば洋服メーカーが「どのサイズの服を何枚作れば何%の人をカバーできるか」という問いに答えられる。正規分布を使って母集団全体を予測するのが統計学の重要な考え方。← S05

🎙 AUDIO — 「出やすい・出にくい」という見方

平均付近の値（μ周辺）は出やすいエリア。μ ± 1.96σ より外側の裾の部分は滅多に起こらない（出にくい）エリア。この対比が第2・3章の推定・検定で活用される。← S05 / S03:p.32

📋 第1章 まとめ
  • 正規分布：左右対称の釣鐘型 / 平均値＝中央値＝最頻値
  • 最重要：分散（σ²）と平均（μ）で形が決まる
  • 分散が大きい → 分布が平べったくなる
  • 分散が決まれば確率が決まる（σ範囲 → 面積が確定）
  • μ ± 1.96σ の範囲に 95% のデータが含まれる（★頻出）
S03:p.33
第2章

統計的推測（1）

学習目標：① 統計的推測の種類について説明できる　② 点推定・区間推定の基本的な概念について説明できる S03:p.36

統計的推測とは
DEFINITION
統計的推測（Statistical Inference）

標本データから母集団分布のパラメータを調べること。S03:p.37

🎙 AUDIO — 用語の整理

「標本データから母集団を推測する」という枠組みは第2回で学んだ全数調査 vs 標本調査の話と同じ。全数調査（国勢調査など）は大規模で実施困難なため、一部を取り出してデータ分析する方法をとる。正規分布に従う母集団なら、形を決めるパラメータは母平均（μ）と母分散（σ²）の2つ。しかし実際には母平均・母分散は不明であることがほとんど。← S05 / S03:p.38-39

図：標本から母集団を推測する統計的推測の枠組み ← S03:p.37-39

統計的推測で「調べること」は2種類ある。S03:p.40-41

  1. 推定：母集団分布のパラメータ自体を求める（例：100人の身長データから日本人全体の平均身長を知りたい）
  2. 検定：母集団分布のパラメータに関する仮説を検証する（例：販売促進企画の効果があったと言えるか？）→ 第3章で詳述
推定：点推定と区間推定
種類 概念 例（母平均の場合） 特徴
点推定 パラメータを1点で推定 「母平均は 0.5」 シンプルだが外れやすい
区間推定 パラメータを区間（幅）で推定 「母平均は 0.3〜0.7 の間」 確率と組み合わせて信頼区間を示せる
🎙 AUDIO — 点推定 vs 区間推定のイメージ

「明日の最高気温は何度？」と聞かれて「25度」（点推定）と答えるよりも、「23〜28度ぐらい」（区間推定）と答えた方が当たる確率が高い、という直感がそのまま当てはまる。← S05

EXAMPLE — ケーキ屋のケーキの重さ（区間推定のイメージ）

あるケーキ屋のケーキは「1個100g」と書かれている。毎日全部買い占めることは不可能なので、毎日3個ずつサンプリングして標本平均を記録し続ける。

  • 3/1：標本平均 98g、3/2：101g、3/3：98g、3/4：105g … と積み上げる
  • 大量のデータが集まると → 正規分布に近づく（中心極限定理）
  • 点推定：最もデータが多い 100g を母平均と推定
  • 区間推定：「99〜102gの間に母平均がある」と推定 → これが95%信頼区間になる
← S05 / S03:p.44-55
中心極限定理（Central Limit Theorem）
DEFINITION
中心極限定理

母集団の分布にかかわらず、標本を多数集めると標本平均の分布は正規分布に近づくという性質。S03:p.53

🎙 AUDIO

ケーキの例では、毎日のサンプル平均（標本平均）をプロットし続けると、やがてそれ自体が正規分布の形になった。この現象が中心極限定理。これによって、母集団の分布が不明でも、標本平均については正規分布の性質（第1章で学んだ面積と確率の関係）を使える。← S05

95%信頼区間：区間推定で使う区間に95%の標本が含まれる場合、その区間を95%信頼区間と呼ぶ。「新しいデータが来ても95%の確率でその区間に収まる」という意味。S03:p.55

📋 第2章 まとめ
  • 統計的推測：標本から母集団のパラメータを調べること
  • 推定の2種類：点推定（1点）/ 区間推定（幅を持って）
  • 検定：第3章で詳述
  • 中心極限定理：標本を多数集めると標本平均の分布は正規分布に近づく
  • 95%信頼区間：95%の標本が含まれる推定区間
S03:p.56
第3章

統計的推測（2）

学習目標：① 統計的仮説検定について結果から正しく結論を導ける　② t検定について概要を説明できる S03:p.59

統計的仮説検定（Statistical Hypothesis Testing）
DEFINITION
統計的仮説検定

標本データのばらつきが偶然かどうかを確率的に調べること。S03:p.60

検定で使う3つの重要用語：S03:p.60-61

用語 英語 定義 ポイント
帰無仮説 H₀ Null Hypothesis 主張したい内容とは逆の仮説 否定されることを期待する仮説
対立仮説 H₁ Alternative Hypothesis 帰無仮説の逆 主張したい内容
有意確率（p値） p-value 帰無仮説が正しい場合に、偶然その値になる確率 p値 < 有意水準 → 帰無仮説を棄却
🎙 AUDIO — 検定のロジック

「主張したい内容を直接証明するのではなく、まずその逆（帰無仮説）を仮定し、それが間違いだと示すことで主張の正当性を示す」という論法を使う。
帰無仮説が「棄却（rejected）される」→ 対立仮説が採択される。
帰無仮説が「棄却されない（保留）」→ 判断がつかない（「主張が正しい」という結論にはならない）。← S05 / S03:p.61

検定のイメージ：くじ引きキャンペーンの例
EXAMPLE — キャンペーン効果の検定

問い：客単価向上のためにくじ引きキャンペーンを実施した。これが客単価に本当に貢献したか？

  • 帰無仮説 H₀：キャンペーンに効果はなかった
  • 対立仮説 H₁：キャンペーンに効果はあった（← 主張したい内容）
S03:p.62
🎙 AUDIO — 検定の手順（前提）

キャンペーン期間外の平均客単価が 500円、標準偏差 50円 で正規分布に従うとわかっているとする。この場合 μ ± 1.96σ = 500 ± 98円 = 402〜598円の範囲に営業日の95%の平均客単価が入る。← S05 / S03:p.63

図：検定のイメージ（570円→保留 vs 620円→帰無仮説棄却）← S03:p.63-65
🎙 AUDIO — 2つのケースの解釈

客単価570円の場合：キャンペーンなしの95%の範囲（402〜598円）内に入っている → 「滅多に起こらないとは言えない」→ 帰無仮説は棄却されない（保留） → 判断つかない。

客単価620円の場合：95%の範囲外（棄却域）に入っている → 「キャンペーンなしでも5%以下の確率でしか起きない」→ 滅多に起こらない → 帰無仮説が棄却される → 対立仮説採択 → 「キャンペーンに効果があった」と結論。← S05 / S03:p.63-65

有意水準・p値・t値
DEFINITION
有意水準（Significance Level）

偶然かどうかを判断する基準となる確率のしきい値。5%（0.05）が慣習的によく使われる。分野によっては1%や0.1%を使うこともある。S03:p.66

🎙 AUDIO — 「5%」の意味

この5%という数字に特別な理由はなく、慣習的に使われているもの。「5%だったら滅多に起こらない基準として使っている」と考えればよい。← S05

両側検定と片側検定の注意：有意水準5%を「両側」で設定する場合は、分布の両端に各2.5%ずつ配置される（合計5%）。「片側5%」とは異なるため、データを見るときは区別する。S03:p.66 ← S05

指標 注目しているもの 判断基準
t値（検定統計量） 横軸（数値） t値が棄却域に入っているか
p値（有意確率） 面積（確率） p値 < 有意水準（例：0.05）か
🎙 AUDIO — t値とp値の関係

t値とp値は注目している軸が違うだけで、結論は同じ。t値が棄却域に入っている ⟺ p値が有意水準（0.05）を下回っている、という対応関係がある。一般的にはp値で議論することが多い。← S05 / S03:p.76-78

t検定（t-test）
DEFINITION
t検定

母分散が不明、かつデータ数が少ない（小標本）場合に使用する検定。t分布を使った検定。S03:p.68

t分布の特徴：S03:p.68-69

  • 標本数が増えれば増えるほど正規分布に近づく
  • 正規分布に比べて95%の幅が広い（より保守的な判断）
  • 自由度 ＝ 標本数 − 1
図：標準正規分布 vs t分布（自由度5）の95%幅の違い ← S03:p.69
🎙 AUDIO — なぜ幅が広いか

t分布は正規分布より少し裾が広い（95%の範囲が ±1.96σ ではなく ±2.57σ 程度）。これはデータが少ないときの不確実性を考慮した、より安全な検定になっている。標本数が増えるにつれ、自由度が大きくなり、t分布は正規分布に近づく。← S05

t検定の例：ケーキの重さ（6個）
PROBLEM

あるケーキ店のケーキは「1個100g」と書かれているが、ある日買った6個の重さは：
102g, 98g, 95g, 104g, 99g, 93g だった。これをもとに「100gではない」と言えるか？S03:p.70

t検定の手順 5ステップ：

  1. 標本データの確認（標本数・標本平均・標本分散）
    標本数 n = 6 / 標本平均 x̄ = (102+98+95+104+99+93)/6 = 98.5g
    標本分散 s² = Σ(xᵢ − x̄)² / (n−1) = 17.1
    🎙 AUDIO — (n−1)で割る理由

    サンプル調査では標本数−1で割る（不偏分散）。これによって「データが1個だけなのに分散が出てしまう」という矛盾を回避できる。← S05

    S03:p.71
  2. 帰無仮説・対立仮説の設定
    H₀（帰無仮説）：重さの母平均は100gである
    H₁（対立仮説）：重さの母平均は100gではない ← 主張したい内容
    S03:p.72
  3. 棄却域の設定（有意水準 0.05 = 5%）
    自由度5（n−1=5）のt分布の両側5%の棄却域：t ≤ −2.57 または t ≥ +2.57
    S03:p.73
  4. 検定統計量（t値）の計算
    FORMULA — t値
    t = (x̄ − μ) / √(s²/n) = (98.5 − 100) / √(17.1/6) ≈ −0.889
    S03:p.74（本科目では公式の暗記不要）
  5. 棄却域とt値を比較し結論
    t値 = −0.889 は棄却域（≤−2.57 または ≥+2.57）の外側にある
    → p値 = 0.415 > 0.05（両側5%の中に入っていない）
    → 帰無仮説は棄却されない（保留）
    → 結論：「100gではない」とは言えない（判断できない）
    S03:p.75-80
⚠️ 重要な注意

帰無仮説が棄却されない場合（保留）は「100gではないとは言えない」であり、「帰無仮説を採択して『母平均は100gである』」という結論にはならない。保留とは「わからない」という意味。S03:p.79-80

図：検定の結論の導き方 ← S03:p.79
🎙 AUDIO — t検定の用途（2標本の場合など）

t検定は1標本だけでなく2標本（2群間の差の検定）でもよく使われる。例：アンケートで男女間に差があるか、2つの練習方法の効果に差があるか。さらに第9回の単回帰分析のモデル評価でも使われる。← S05 / S03:p.85

📋 第3章 まとめ
  • 帰無仮説が棄却される → 対立仮説が採択される
  • 帰無仮説が棄却されない（保留） → 対立仮説とは言えない（判断できない）
  • t検定の流れ：① 標本データ確認 → ② 帰無仮説・対立仮説設定 → ③ 棄却域設定 → ④ 検定統計量（t値）計算 → ⑤ 比較 → ⑥ 結論
  • t値（横軸に注目）とp値（面積に注目）は注目する軸が違うだけで結論は同じ
S03:p.81
第4章

EXAMPLE — 東京の気温は本当に昔より暑くなったか？

検証内容：1975〜1989年（古い15年）と 2006〜2020年（新しい15年）の東京都の平均気温に有意な差があると言えるか？← S05 / S03:p.87

  • 帰無仮説 H₀：平均気温に違いはない
  • 対立仮説 H₁：平均気温に違いがある（← 主張したい内容）
  • 手法：2標本t検定（等分散を仮定しない）
  • 有意水準：α = 0.05

🌐 外部補足

Welchのt検定について：スライドでは「等分散を仮定しない2標本によるt検定」と表記されているが、これは統計学でWelchのt検定と呼ばれる手法。等分散を仮定しない場合でも使用でき、等分散の場合にも適用できるため汎用性が高い。本来の手順はF検定で等分散かどうかを先に確認してからt検定の種類を選ぶが、実務では先にF検定をせずWelchのt検定をデフォルトで選ぶことが多い。自由度の計算式も通常のt検定（データ数−1）とは異なり、より複雑な式（Welch-Satterthwaite近似）を使う。S03:p.86 + 外部補足

統計やデータサイエンスに関する資格試験（ITパスポート、統計検定など）でも検定結果の表の見方が出題されることがある。← S05

目的 使用する検定
母平均に関する検定（大標本・母分散既知） z検定
母平均に関する検定（小標本・母分散不明） t検定

📚 補足B: 背景知識
  • 正規分布の確率密度関数は覚えなくてよい（授業で明言）。式中にσ²とμが含まれている点だけ押さえる。S03:p.17
  • 標準偏差σ = √σ²（分散の正の平方根）。スライドでは以降σと略記。S03:p.17
  • 標本分散の計算はΣ(xᵢ−x̄)²をn−1（標本数−1）で割る（不偏分散）。全数調査の分散と異なる点に注意。S03:p.71
  • Welchのt検定：「等分散を仮定しない」t検定。等分散の場合にも使用可。自由度の計算式が通常と異なる。S03:p.86 + 外部補足
  • F検定：2群の分散が等しいかを検定する手法。t検定の種類を選ぶ前に本来実施する。S03:p.86
  • χ²検定：母分散に関する検定。今回は範囲外。S03:p.86
  • ノンパラメトリック検定：正規分布を前提としない場合の検定群。S03:p.86
🎯 補足C: 試験・レポートで重要な観点
  • 帰無仮説が保留＝「帰無仮説の採択」ではない。「わからない」という意味。S03:p.79-80
  • p値 < 有意水準（0.05）→ 帰無仮説を棄却 → 対立仮説採択。S03:p.76-78
  • 1.96σ → 95%（正確に95%となる境界値）を即答できるように。S03:p.27
  • 検定結果の表の読み方：T Stat（t値）・P two-tail（p値）・t Critical two-tail（棄却域境界）の3点。S03:p.88-89 ← S05
  • t分布が正規分布より95%の幅が広い（±1.96σ → ±2.57σ）理由：データが少ないときの不確実性を考慮するため。S03:p.69
  • 統計・データサイエンス関連の資格試験でも検定結果の表の見方が出題される。← S05
