第1章

モチベーションコントロールとは何か

第1章 学習目標

  1. モチベーションについて理解する

モチベーションとは何か

【定義】モチベーション

ものごとを行うときの動機や意欲になるもの。刺激や熱意など。やる気。 ← S03:p.6

🔊 AUDIO SUPPLEMENT

「モチベーションが上がらない」「モチベーションが上がってきた」のように日常的に使われる言葉。どんな時に自分のやる気が出てくるのか——それがモチベーションという言葉に秘められた意味。 ← S05-A

やる気の源としては、「人から褒められる」「十分な準備ができたと感じた時」「人から頼まれる」「俄然（がぜん）やる気が出るよう命じられる」など、人によって異なる。 ← S05-A

モチベーションコントロールとは何か

【定義】モチベーションコントロール

変えられることと変えられないことの違いを理解し、自分の考え方や行動にプラスのエネルギーを保つこと。 ← S03:p.7

言い換えると：プラスのエネルギーを保ち続けること。 ← S03:p.7

🔊 AUDIO SUPPLEMENT

やる気は何か特別な出来事があると上がったり下がったりする。人生観・価値観・育った環境・人間関係など、変えられるものと変えられないものの両方がある。それらを踏まえた上で自分の考え方や行動にプラスのエネルギーを維持・保持することが「コントロール」の本質。 ← S05-A

やる気を削ぐ阻害要因の例：努力を評価されない、手柄を横取りされる、成果をネガティブに評価される。自分のモチベーションが下がる傾向がどういう場合かをよく知っておくことが重要。 ← S05-A

モチベーションコントロール — ファーストステップ：現状分析

FACT現状分析が最初のステップ。 ← S03:p.8

1

やる気のもとは何か ← S03:p.8

何からやる気が生まれやすいか——刺激・賞賛・準備完了感・他者からの依頼など、人によって異なる。

2

やる気の阻害要因は何か ← S03:p.8

どういう状況でモチベーションが下がるかを把握する。

3

モチベーションの波を考える ← S03:p.8

感情のムラ・波は人によって大きく異なる。いつ高く、いつ低いか——その傾向を理解することがスタート。

セカンドステップ：目標設定

FACT具体的で明確な目標設定が必須。 ← S03:p.9

ポイント内容
魅力的な目標達成して嬉しくない目標はやる気が起きない ← S03:p.9
自分が納得できる目標押し付けられた・無理な・納得できない目標は達成しようと思わない ← S03:p.9
ギャップの把握現状と目標のギャップを理解する ← S03:p.9
具体的ステップ目標に到達するための段階的ステップを考える ← S03:p.9

ラストステップ：目標達成

FACT目標達成がモチベーションコントロールの完成形。 ← S03:p.10

1

目標の細分化 ← S03:p.10

大きな目標を細切れの小さな目標に分解する。達成感を何度も味わうことができ、モチベーション維持につながる。

2

将来設計との接続 ← S03:p.10

小さな目標を達成するたびに、それが自分の将来に向かう一歩であると感じられれば、努力・苦労もポジティブなエネルギーとして維持できる。

3

妥協ではなく前向きな軌道修正 ← S03:p.10

「本当はやりたかったのに妥協して…」という気持ちのままでは下がる。必要に応じて現実的・前向きに軌道修正することが大切。

4

意味付けと次の目標 ← S03:p.10

目標自体を明確化し「社会貢献できる」等の意味付けをする。達成したらそこで満足せず、すぐ次の目標を考える。

第1章まとめ

  1. 自分の考え方や行動にプラスのエネルギーを保つことがモチベーションコントロールの目的である ← S03:p.11
  2. まず自分の現状分析をしっかりと行うことが大切（ファーストステップ） ← S03:p.11
  3. 具体的で明確な目標設定が必須（セカンドステップ） ← S03:p.11
  4. 目標達成が最も効果が高い。達成したら次の目標を立てていく ← S03:p.11
第2章

モチベーションを高く保つ考え方

第2章 学習目標

  1. モチベーションを高める2つのアプローチを理解する
  2. 目的志向型と問題回避型のどちらもモチベーションが高い人は存在することを理解する
  3. モチベーションサイクルを理解する

モチベーションを高める2つのアプローチ

項目 目的志向型 問題回避型
特徴 目的意識が明確。仕事は「やりたいことを実現するための手段」。ゴールに向かってまっすぐ進む ← S03:p.15 問題が起きないよう慎重に考える習慣がある。問題回避に大きなエネルギーを費やす ← S03:p.16
多い職種 短期目標の達成を求められる仕事（例: 営業） 正確さを常に求められる仕事（例: 会計・品質管理）
他者からの見え方 — 目的志向型からは「ブレーキ・障害」に見えることがある（誤解）

⚠️ 重要な注意：目的志向型 = ポジティブで仕事ができる人、問題回避型 = ネガティブで仕事ができない人、という決めつけは誤り。
どちらも異なる思考パターンであるだけ。両方を使い分けることが大切。 ← S03:p.17

🔊 AUDIO SUPPLEMENT

日本人は特に問題回避型のアプローチを取る人が多い傾向がある。目的思考型と問題回避型は対立的な立場にあると考えがちだが、それは誤解。 ← S05-A

モチベーションを高めるポイント — 目的志向型

  • 短期目標の達成を求められる仕事に多い ← S03:p.18
  • 成功イメージを持ち、そのために何が必要か想像を膨らませる ← S03:p.18
  • 成功を阻害する要因は何かを考え、過去の失敗事例を分析する ← S03:p.18

モチベーションを高めるポイント — 問題回避型

  • 正確さを求められる仕事に多い ← S03:p.19
  • 問題の本質について徹底的に議論し、問題点を整理する。問題は問題として淡々と受け止める ← S03:p.19
  • 次に理想のカタチをイメージして言葉に表し、問題解決後の状況を具体的にイメージする ← S03:p.19
🔊 AUDIO SUPPLEMENT

問題回避型は「毎回ネガティブ」ではない。成功を阻害する要因に気づくからこそ、その問題点を解決する答えを出せれば成功確率が非常に高まるタイプ。問題を把握したら理想の形を明確に言葉にし、問題解決後の状況を具体的にイメージするプロセスで目標を達成していく。 ← S05-A

モチベーションコントロールのコツ

PROPOSAL今、自分はどの思考パターンになっているかを把握して対処すること ← S03:p.20

自分の状態対処方法
目的志向パターンと自覚 さらに成功イメージを膨らませていく ← S03:p.20
問題回避パターンと自覚 問題の本質を淡々と受け止め、問題解決後の理想的な状況を考える。そこで立ち止まらず先に進む ← S03:p.20

モチベーションサイクル

【定義】モチベーションサイクル

努力 → 成果 → 報酬 → 満足 → （さらに努力へ）という正のフィードバックループ。このサイクルを回し続けることがモチベーションを高く維持する基本。 ← S03:p.21

図1: モチベーションサイクル（S03:p.21 再現）
🔊 AUDIO SUPPLEMENT

具体例（スポーツ）：技術を磨く（努力）→ 試合でシュートが決まる（成果）→ レギュラー選出・出場機会増加（報酬）→ 満足度が高まる（満足）→ さらに努力したくなる。 ← S05-A

報酬は金銭的なものに限らない。自分なりの様々な報酬を見つけて感じることが重要。 ← S05-A

第2章まとめ

  1. モチベーションを高めるには2つのアプローチ（目的志向型・問題回避型）がある ← S03:p.22
  2. 多くの人はこの両方を使い分けることが多い ← S03:p.22
  3. 目的志向型は成功イメージを膨らませると効果的 ← S03:p.22
  4. 問題回避型は問題解決後の理想を考えると効果的 ← S03:p.22
  5. モチベーションサイクルを回すことが大切 ← S03:p.22

ビジネススキル開発の考え方

第3章 学習目標

ビジネススキルとは何か

【定義】ビジネススキル
  • 業務を行うために必要な知識やノウハウ ← S03:p.26
  • 職場の人間や顧客との円滑な関係を築くこと ← S03:p.26
  • 問題を把握し、それを解決していくこと ← S03:p.26
🔊 AUDIO SUPPLEMENT

学校ではあまり専門的に学ぶ機会がないが、書店のビジネス書コーナーが大きなスペースを占めていることからも分かるように、学びたいという意欲は高い。人と人をつなぎながら新しいものを作る、価値観を理解してもらう、製品やサービスを提供するという一連の行動の中で必要なスキルがビジネススキル。 ← S05-A

ビジネススキルの3分類

図2: ビジネススキルの3分類（S03:p.27 再現）

テクニカルスキルとは何か

【定義】テクニカルスキル

業務を行うために必要な知識やノウハウ。仕事の環境・会社規模・業界によって必要な内容が異なる。 ← S03:p.28

種類具体例
業界知識・競合情報・商品知識競合他社の強み・弱点、自社製品の課題 ← S03:p.28
その他専門知識部品調達方法、顧客が求めるサービス ← S03:p.28
パソコン・スマートフォンスキル各種ソフトウェア ← S03:p.28
語学スキル英語（グローバルビジネスの共通語）、中国語・フランス語・ドイツ語等 ← S03:p.28
プレゼンテーションスキル相手に分かりやすく伝える方法 ← S03:p.28
マネジメントスキルチームマネジメント・リーダーシップ ← S03:p.28
🔊 AUDIO SUPPLEMENT

ビジネスは今やボーダーレス・グローバルに展開している。英語は共通語として必須だが、中国語・フランス語・ドイツ語など特定の国の言葉を話せることがプラスになることもある。 ← S05-A

ヒューマンスキルとは何か

【定義】ヒューマンスキル

職場の人間や顧客との円滑な関係を築くこと。 ← S03:p.29

  • ビジネスマナー、タイムマネジメント、モチベーションコントロール、コミュニケーション等 ← S03:p.29
🔊 AUDIO SUPPLEMENT

名刺の渡し方・個室での座り方・会議の議事進行など、様々な場面でビジネスマンとしての常識が問われる。海外の人と付き合うとなればさらに高度化し、言葉を知るだけでは不十分になる。コミュニケーションスキルが高い人は仕事で成功する確率が大変高い。プライベートの時間（趣味・家族との時間）でもヒューマンスキルを磨く機会がある。 ← S05-A

コンセプチュアルスキルとは何か

【定義】コンセプチュアルスキル

問題を把握し、それを解決していくこと。論理的思考、プロブレムソルビング（Problem Solving）、クリティカルシンキング（Critical Thinking）等。 ← S03:p.30

🔊 AUDIO SUPPLEMENT

物事を論理的に捉え、論理的に説明できるか。チームの目標を設定・達成する際にどれだけ戦略的な考え方を持てるか。管理職・マネジメントの仕事をするようになるほど求められる。 ← S05-A

ビジネススキルを開発するヒント

方法具体的な手段
知識から得る 書籍・ビジネスコラム（インターネットでも可）。読み物から知識を継続的に吸収 ← S03:p.31
アウトプットも重要 関心ある分野の勉強会・セミナーに参加。講師に質問、意見を発表。インプットとアウトプットのバランス ← S03:p.31
人脈から得る 社内人脈・社外人脈からロールモデル（お手本）を探す ← S03:p.31
専門家から個別対応で得る コーチング（継続的な気づきのコミュニケーション）、カウンセリング（相談）を活用 ← S03:p.31

ビジネススキルの実践（キャリア段階別）

新入社員・若手社員の場合 ← S03:p.32

  • 基本的な業務知識・ビジネスマナー等、テクニカルスキルをより多く身につける
  • 報告・連絡・相談（ホウレンソウ）のような基本的なコミュニケーションスキルが求められる
  • 先輩のやり方を真似することが近道

先輩社員・管理職の場合 ← S03:p.33

  • 傾聴力など、より高度なコミュニケーションスキルが求められる
  • 交渉・難しい問題の説明など高度なコミュニケーション能力が必要。実践・失敗・反省で磨く
  • リーダーシップや戦略立案などのコンセプチュアルスキルに注目する
  • 優秀な人のやり方を見ることが最も近道

顧客・取引先との関係 ← S03:p.34

  • 高度なコミュニケーションスキル、深い経験・知識に基づくビジネスの提案力が期待される
  • 異文化対応力や、異分野とのコラボレーションを提案する意外性のある対応が求められる

第3章まとめ

  1. ビジネススキルは大きく分けて3つ：テクニカルスキル・ヒューマンスキル・コンセプチュアルスキル ← S03:p.35
  2. 開発するには知識から得る方法・人脈から得る方法・専門家から個別対応で得る方法がある ← S03:p.35
  3. 自分のステージや付き合う相手によって求められるスキルのウェイトが異なる ← S03:p.35

情報発信の考え方

第4章 学習目標

  1. 情報発信の目的を理解する

  3. 情報発信が失敗する理由を理解する
  4. 情報発信にインターネットが果たす役割を理解する

情報発信とは何か

【定義】情報発信

自分が提供したい情報を色々なメディアを活用して、継続的に伝えていく行動。 ← S03:p.39

⚠️ 情報の内容次第で、相手との信頼関係を築けることもあれば、逆に信頼を失うこともある。慎重かつ積極的に行う必要がある。 ← S03:p.39

🔊 AUDIO SUPPLEMENT

情報を発信するという行動自体が、自分のビジネススキルを開発することにもなる。また、継続的に自分を人に知ってもらうことでパーソナルブランドを築く際にも大変有効。 ← S05-A

情報発信の4つの目的

図3: 情報発信の4つの目的（S03:p.40 再現）
#目的内容
①認知自分は何者か、伝えたいことを相手に知らせる ← S03:p.40
②共感相手に気に入ってもらう、好印象を与える ← S03:p.40
③参加相手に行動に移してもらう ← S03:p.40
④率先感動や満足によって、強く勧めてもらう（口コミ拡散） ← S03:p.40

ビジネスにとって情報発信が大切な理由

FACT今の時代、買い手は売り手から「売り込まれること」をとても嫌がる。 ← S03:p.41

→ 買う前に売り手と直接コンタクトすることを嫌がる（売り込まれると警戒するため）

→ よって、買ってもらう前に情報発信することが大切 ← S03:p.41

🔊 AUDIO SUPPLEMENT

買い手は事前に情報を収集して自分で判断したい。売り手からの売り込みを受ける前に意思決定したい。だからこそ私たちは相手が行動を取るずっと前から継続的に情報を発信していく必要がある。 ← S05-A

情報発信が必要になったきっかけ：インターネットの普及

図4: ネット普及による主導権の移行（S03:p.42 再現）
  • 以前：買い手は情報を自由に得られなかった。売り手が情報を握っていた ← S03:p.42
  • 現在：ネットの普及で、主導権が売り手から買い手に移行した ← S03:p.42
  • → だから情報発信を日頃からしなければならない ← S03:p.42

情報発信が失敗する3つの原因

#原因説明
① 受け手の視点が足りない 顧客が何を求めているかの研究・配慮が不足している ← S03:p.43
② 自分に都合のいい情報だけを発信 トラブル・ネガティブ情報を隠すと信頼が高まらない。今はあらゆる形で商品・サービスが比較される時代 ← S03:p.43
③ 相手がどう思うかの想像力が足りない 言葉遣い・情報量・頻度が与える感情的影響を考慮しないと「嫌われる情報提供者」になる ← S03:p.43

情報発信にインターネットを活用する

ツール特徴適した情報
ホームページ 比較的固定した情報を見せる ← S03:p.44 プロフィール・会社概要

📚 補足B: 知っておくべき背景知識

① モチベーション古典理論との対応

本講義の「プラスのエネルギーを保つ」というモチベーション観は、以下の古典理論と深く関連する。

理論提唱者・年内容本講義との接続
欲求階層説
[外部補足・確認済: Psychological Review, 1943]		 Abraham H. Maslow
1943年（Psychological Review誌掲載）		 生理・安全・愛・尊重・自己実現の5段階の欲求が階層を成し、下位が充足されると上位が動機となる 「目標を達成したら次の目標を立てる」サイクルは自己実現欲求への継続的な接近に対応。「何度も達成感を味わう」細分化手法は段階的な欲求充足のプロセスとも整合する
二要因理論（動機付け・衛生理論）
[外部補足・確認済: The Motivation to Work, 1959]		 Frederick Herzberg
1959年（著書 "The Motivation to Work" 発表）		 職場における満足要因（動機付け要因：達成・承認・成長など）と不満足要因（衛生要因：給与・労働条件など）は別の軸。衛生要因を改善しても満足は生まれず、動機付け要因があって初めて内的モチベーションが高まる 「達成感・承認・成長」を重視するモチベーションサイクルの「報酬」概念は、金銭的報酬だけでなく動機付け要因としての内的報酬が重要という Herzberg の知見と一致。「評価されないとやる気が下がる」という阻害要因の説明も動機付け要因の欠如に対応する

⚠️ マズローの欲求階層ピラミッド図はマズロー本人が描いたものではなく、後世の研究者によって図式化されたもの（マズローの1943年論文に図は存在しない）。

② ビジネススキル3分類の理論的背景（Katz, 1955）

[外部補足・確認済: Harvard Business Review, 1955 / ERIC EJ104214]
Robert L. Katz は1955年、Harvard Business Review 誌（Vol.33 No.1, pp.33–42）に論文「Skills of an Effective Administrator」を発表し、管理者に必要なスキルをテクニカル・ヒューマン・コンセプチュアルの3分類に整理した。同論文は1974年に "HBR Classic" として再掲載されるほど影響力が大きく（Harvard Business Review, Vol.52 No.5, pp.90–102）、今日の経営学・リーダーシップ教育の基礎として広く参照されている。

スキルKatz の定義重要度の変化（キャリア段階）
テクニカルスキル 特定の業務・活動に関する専門知識と習熟度 新人・若手で最も重要。上位職ほど比率が低下
ヒューマンスキル 他者と効果的に連携するための対人・内省スキル 全階層で一貫して重要（最も普遍的）
コンセプチュアルスキル アイデアや概念を用いて思考する能力（戦略・問題解決） 上位管理職ほど比重が増大

講義の「新入社員はテクニカルスキル重視 → 管理職はコンセプチュアルスキル重視」という説明は Katz 理論の主要命題に直接対応する。

③ インバウンドマーケティングと情報発信

[外部補足・確認済: HubSpot公式サイト / Wikipedia: Brian Halligan]
講義で説明された「買い手が事前に情報収集し、売り手からの売り込みを避ける」という現象は、マーケティング分野でインバウンドマーケティング（Inbound Marketing）と呼ばれる概念に対応する。
Brian Halligan と Dharmesh Shah が2004〜2006年のMIT在籍中に概念化し、2006年6月にHubSpotを共同創業して提唱・普及させた。「売り込む（outbound）」のではなく「価値ある情報を継続発信して顧客を引き寄せる（inbound）」という発想の転換が核心である。インターネット普及による情報の非対称性の解消（売り手から買い手への主導権移行）がこの変化の背景であり、本講義第4章の主張と完全に一致する。

アウトバウンド（従来型）インバウンド（現代型）
アプローチ売り手 → 買い手へ押しつける情報を発信して買い手を引き寄せる
買い手の態度受動的・回避傾向能動的に情報収集・自己判断
主導権売り手が情報を握る買い手に主導権が移行
本講義との対応「売り込まれることを嫌がる」状況「買う前に情報発信することが大切」

🎯 補足C: 試験・レポートで重要な観点

定義の正確な記述（必須）

  • モチベーションコントロールは「やる気を高めること」ではなく「プラスのエネルギーを保ち続けること」——維持・保持の概念が重要
  • 目的志向型 ≠ ポジティブな人、問題回避型 ≠ ネガティブな人——この誤解を解消できているかが問われやすい
  • ビジネススキルの3分類はテクニカル・ヒューマン・コンセプチュアル。混同しないこと

モチベーションコントロールの3ステップ

現状分析（ファースト）→ 目標設定（セカンド）→ 目標達成（ラスト）の順序と内容を正確に説明できること。特に「目標の細分化」「前向きな軌道修正」「意味付け」がラストステップの要点。

情報発信の4目的

認知 → 共感 → 参加 → 率先 の順序と各目的の意味を記述できること。

情報発信失敗の3原因

①受け手の視点不足 ②自分本位の発信 ③想像力不足——3つを具体例と共に説明できること。

よく出る比較問題

  • ホームページ vs ブログ vs ソーシャルメディアの違い（ストック情報 vs フロー情報）
  • 目的志向型と問題回避型のそれぞれのモチベーション向上のポイント
  • 若手社員 vs 先輩・管理職で求められるスキルの違い

  • ヒューマンスキルとして挙げられた「タイムマネジメント」「モチベーションコントロール」は本科目全体のテーマと連動

関連する外部概念（コンセプチュアルスキル）

クリティカルシンキング（Critical Thinking / 批判的思考）：コンセプチュアルスキルの中核。情報・前提・論理の妥当性を系統的に評価・検証する思考法。「本当にそうか？」「前提は正しいか？」を問い続ける姿勢がビジネスの意思決定・問題解決に不可欠。 [外部補足]

プロブレムソルビング（Problem Solving / 問題解決思考）：コンセプチュアルスキルの実践形態。①問題の定義 → ②原因分析 → ③解決策立案 → ④実行 → ⑤評価 のプロセスを踏む。MECE（モレなくダブりなく）やロジックツリーなどのフレームワークが整理ツールとして広く使われる。 [外部補足]

関連する外部概念（情報発信・マーケティング）

コンテンツマーケティング（Content Marketing）：インバウンドマーケティングの主要手法。有益なコンテンツ（記事・動画・ポッドキャスト等）を継続的に発信することで見込み顧客を引き付けるアプローチ。本講義「ブログ・ホームページ・SNSの活用」「動画・写真は強力なツール」という説明はコンテンツマーケティングの実践に直接対応する。 [外部補足]

エンゲージメント（Engagement）：情報発信において受け手が「反応・共感・行動」すること。講義の「参加を促す（③）→ 率先して広めてもらう（④）」という2段階は、マーケティングで言うエンゲージメントとアドボカシー（推奨者化）に対応する。 [外部補足]

本講義の位置づけと今後の学習へ

  • 本回は「セルフマネジメント論」の後半の核心テーマ。自己管理（モチベーション）→ 能力開発（ビジネススキル）→ 社会との接続（情報発信）という個人の内側から外側への一連の枠組みを構成している
  • Katz理論（1955年）が示すように、キャリアが進むほどコンセプチュアルスキルの比重が増す。学生のうちからテクニカル・ヒューマン両スキルを意識的に磨くことが長期的なキャリア開発の基盤となる
