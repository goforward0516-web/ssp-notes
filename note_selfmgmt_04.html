コミュニケーションとは何か

用語意味（広辞苑）
会話2人、あるいは少人数で向かい合って話すこと
対話向かい合って話すこと。相対して話すこと
← S03:p.6
DEFINITION — コミュニケーション（広辞苑）

社会生活を営む人間の間に行われる知覚・感情・思考の伝達手段

← S03:p.7
🎙 AUDIO SUPPLEMENT

コミュニケーションとは「言いっぱなし」にすることではない。自分の思いが相手に伝わっているところが重要なポイント。言葉だけでなく、視覚・聴覚（表情・ジェスチャー・声のトーン）に訴えることで、より豊かに感性・気持ちを伝えられる。

← S05

コミュニケーションの目的

  • 自分の考えを相手に伝えること
  • 自分の思い通りに相手を動かすこと
← S03:p.8
🎙 AUDIO SUPPLEMENT

「相手に分かってもらう」だけでなく、その上で「自分の思うように相手に動いてもらう」ことが大きな目標になる。

← S05

コミュニケーションの心構えとコツ

心構え ← S03:p.9

  • 心を開いて相手と関わること
  • 相手の立場に立って働きかけること
  • 相手の眼を見て思いを訴えること
  • 言葉以外の表現方法も使うこと（情熱・ジェスチャー・声の大きさやトーン）
  • 言葉と態度、その両方を使うこと

コツ ← S03:p.10

  • 「自分の話」を冷静に聞く、もう一人の自分を持つこと
  • 相手に対する尊敬の念を持つこと
  • 自分の言葉で語ること（他人の言葉を真似ない）
  • 言葉の強弱や抑揚も効果的
🎙 AUDIO SUPPLEMENT

「もう一人の自分」とは、自分が話している内容をどこか客観的に聞いている自分のこと。「今の話し方で伝わったか？」「相手は誤解していないか？」「言い方がきつくなかったか？」を振り返る習慣が大切。

← S05

ビジネスコミュニケーション

目的 ← S03:p.11

  • 自分の考えを相手に伝えて、納得してもらうこと
  • 自分の思い通りに相手を動かし、ものごとを先に進め、より早く確実に成果につなげること

特徴 ← S03:p.12

KEY POINT

ビジネスコミュニケーションでは、普通のコミュニケーションより論理的なメッセージを送る必要がある。相手を説得できることが重要。コミュニケーションの質がビジネスの成功の原点になる。

セルフチェック項目 ← S03:p.13

  • 「話しやすいタイプ」であるか
  • 「ひとりよがりな話し方」をしていないか
  • 「適切な言葉遣い」ができているか
  • 相手に対する「心遣い」「気働き」が重要

成功のためのコツ ← S03:p.14

  • 表現に注意し、摩擦をなくすこと
  • 相手の心の動きを読むこと
  • 状況を察知して、素早く対応すること
  • 思い込みを排除すること
  • 自分中心ではなく、相手中心で考える（＝顧客視点）
🎙 AUDIO SUPPLEMENT

ビジネスコミュニケーションを成功させる人物像として、「とっつきにくそうなのに人から好かれる人」がいる一方、「明るく愛想がいいのに嫌われる人」もいる。ビジネスでは外見・趣味よりも「心配り・気働き」がお互いに交換できるかどうかが重要。価値観が全く違う人ともいい仕事ができることがある。

← S05

第1章まとめ ← S03:p.16

  • コミュニケーションは言葉だけでなく、視覚や聴覚にも働きかけるのが効果的
  • 効果的なコミュニケーションは、相手の立場に立ち、言葉以外の表現方法も使うことが重要
  • ビジネスコミュニケーションは、論理的なメッセージで相手を説得する必要がある
  • ビジネスコミュニケーションが成功するには、相手への「心遣い」「気働き」が鍵になる
CHAPTER 02

傾聴とは何か

第2章 学習目標
  • 傾聴のメリットや効果を理解する
  • 7つの傾聴スキルを理解する
  • コーチングとは何か理解する

傾聴の定義・目的・メリット

DEFINITION — 傾聴（けいちょう）

カウンセリングやコーチングにおけるコミュニケーションスキルの一つ。
注意を払って、より深く丁寧に相手の話に耳を傾けること。相手の話を共感的な態度で真摯に聞く技法のこと。

← S03:p.20

傾聴の目的 ← S03:p.21

  • 自分や相手を客観視することで緊張をなくし、ストレスをコントロールできるようになること
  • 聴く力、人と関わる力、コミュニケーション力をつけること

傾聴力を身につけるメリット ← S03:p.22

  • 緊張せず、リラックスして相手と関われる
  • 相手の気持ちを汲む力が上がり、信頼関係を築きやすくなる
  • 質問力がつき、状況を客観視できるようになる
  • 感情を深く観察できるため、ストレスが減る
🎙 AUDIO SUPPLEMENT

傾聴を意識し出すと、自分の中で緊張感が収まってきてリラックスできる。相手を理解できると「分かってもらっている」という感覚が生まれ、相互の信頼関係に育っていく。対立的な状況の中でも、お互いが今どのように感じているかを考察する術が身につき、ストレスが徐々に減っていく。

← S05

傾聴をする際の注意点

NG行動説明
相手の話を遮らない 自分の思いを先に言いたい衝動、相手の話がもたもたしてイライラする場合も我慢する
相手の話を盗まない 相手が言おうとしていたことを横取りして、まるで自分が気づいたかのように言わない
相手の話を否定しない 言葉だけでなく、表情で「不快」「違う」を見せることも含む。無意識にやってしまうことが多い
← S03:p.23
🎙 AUDIO SUPPLEMENT

難しそうな顔をするだけで、相手は「懸念を感じている」「信用していないのでは」とすぐ察知する。人間の表情は豊かなので、無意識のうちに否定の表情を出していることがある。

← S05

傾聴の7つのスキル

← S03:p.24
#スキル意味・使い方注意点
① うなずき 相手の話す音が聞こえていることの意思表示。相手を安心させる。
楽しい話：浅く早く　つらい話：深くゆっくり		 うなずきのペースと深さを話題に合わせる
② あいづち 相手の話す内容が伝わっていることの意思表示。「なるほど」「そうですか」等。何種類か使い分ける。 ネガティブなあいづち（首をかしげる等）は避ける
③ アイコンタクト 相手の話す気持ちが伝わっているという意思表示。「目は口ほどに物を言う」。優しい眼差しを送る。 アイコンタクトがないと相手はパニックする（「面白くないのか」「不快なのか」と思う）
④ 繰り返し 相手の言葉をそのまま（言葉を変えずに）返す（オウム返し）。主語・目的語も全て繰り返す。 言葉を変えるとニュアンスが変わる。部分的に繰り返さない。
⑤ 言い換え 相手の話をまとめて要約すること。相手は「理解されている」と確認できる。 先走り・ミスリードしないこと。感情の言い換えは不適切（繰り返しで対応）。
⑥ うながし あいづちとセットで使う。「それから？」「もっと言うとどういうことになるんだろう」等で話を深める。 話が一瞬止まったとき・内容を深く考えてもらうときに効果的
⑦ 沈黙 相手が心の深い部分を探っている状態。沈黙から戻るのをじっと待つ姿勢が基本。 話しかけるのはむしろマイナス。相手の思考を遮り、混乱させる可能性がある。
← S03:p.25-31

コーチングとは

DEFINITION — コーチング

相手の心の中に隠れているもの、潜在的に存在するものを顕在化させる（表に出す）コミュニケーションのアプローチ。

← S03:p.32-33
🎙 AUDIO SUPPLEMENT

コーチが教えるのは知識ではない。コーチがきっかけを与えることで、本人が自ら気づくことをサポートする。そのプロセスに傾聴が大変役に立つ。

← S05

第2章まとめ ← S03:p.34

  • 傾聴は、相手の自発性を大切にしたコミュニケーションの手法
  • 人材育成、マネジメント、交渉などにおいて効果が期待できる
  • 傾聴には7つのスキルがあり、各段階の目的に沿って使い分けると効果がある
  • コーチングとは「傾聴」を使ったコミュニケーションの手法であり、相手の心の中に潜在しているものを顕在化させる効果が期待できる
CHAPTER 03

アサーションとは何か

第3章 学習目標
  • アサーションについて理解する
  • アサーティブのメリットや、それを阻害する原因を理解する
  • 問題をアサーティブに解決するスキルを理解する

アサーションの定義と位置づけ

DEFINITION — アサーション（Assertion）

人は誰でも自分の意見や主張を表明する権利があるという立場に基づく適切な自己主張のこと。自分の主張を押し付けるのでも、一方的に我慢するのでもなく、お互いを尊重しながら率直な自己表現をすること。

← S03:p.38

アサーション表現の原則 ← S03:p.40

KEY POINT — アサーション表現の原則

「私はこう考えている」→「だからあなたにはこうしてほしい」
自分の意見をベースに置いて、そこから相手へのリクエストをする。
相手へのリクエストだけを言うと威圧的・押しつけがましく聞こえてしまう。

アサーションによる気づき ← S03:p.41

  • 人は皆考え方が違う
  • 自分の考えや感じ方は自由に表現してもいい
  • 立場や能力を超え、人は皆対等である（部長も若手も表現の自由は同じ）
  • 相手に「NO」といっても信頼関係は維持できる
  • 選択肢は自分で選んでいる実感が持てる
  • 等身大の自分でいても大丈夫という思いが広がる
🎙 AUDIO SUPPLEMENT

本音と建前を使い分けすぎると、相手からむしろ信頼されなくなってしまう。NOと思った時には言うべき。等身大の自分でいて、選択肢を自分で選んでいるという実感が「アサーティブな状態」。

← S05

アサーティブになれない4つの原因

← S03:p.42
#原因説明
① 自分の気持ちが十分に把握できていないこと 自分自身のことがよく分かっていないと、相手にうまく伝えられない
② 周りからの評価や結果を気にしすぎていること 評価・結果への過度な意識が、本音を伝えようとする気持ちを妨げる
③ 非言語的表現が使用する言語と矛盾していること やりたくないのに「やりたい」と言っているような矛盾した状況では自己主張が難しい
④ 相手を大切にしたり相手からも大切にされていないこと お互いへの尊敬の念がないと、自分の思ったことを伝えられない

問題をアサーティブに解決するスキル — DESC法

← S03:p.43-47
ステップ英語内容注意点
D — 描写する Describe 客観的・具体的な事実、特定の事柄や言語そのものを描写する 相手の動機・意図・態度を描写しない。あくまで事実関係のみ。
E — 表現・説明・共感 Express / Explain / Empathize 相手の特定の事柄や言動に対する自分の気持ちを、建設的かつ明確に伝える 曖昧にならないよう工夫する
S — 特定の提案 Specify 相手に望む行動や妥協案、解決策などについて、より具体的で現実的な提案をする 抽象的・曖昧な言い方をしない
C — 選択する Choose あらゆる結果を想定し、具体的・現実的で実現可能な行動案の選択肢を示す 実現できないことを示しても相手は困惑するだけ

第3章まとめ ← S03:p.48

  • アサーションとは、より良い人間関係を築くためのコミュニケーションスキルの一つ
  • アサーティブであることによって自分に正直に生き、かつ相手の意思を尊重することができる
  • 問題をアサーティブに解決するにはDESC法が効果的
CHAPTER 04

人を動かすコミュニケーションとは

第4章 学習目標
  • 人を動かすコミュニケーションの特徴を理解する
  • 信頼関係を築くコミュニケーションの手法を理解する
  • 相手が動きやすくなるコミュニケーションを理解する

人を動かす3段階プロセス

← S03:p.52
🎙 AUDIO SUPPLEMENT

第2段階まで（理解・納得）は大変骨が折れるが、第3段階（行動への移行）は「軽く背中を押す」程度でよい。理解・納得した状態でも実際に行動に移してもらわなければ「動いた」とは言えない。

← S05

人を動かすコミュニケーションの工夫 ← S03:p.53

  • 肯定的な表現を使う（ネガティブな表現はしない）
  • 相手のプラス面をとりあげる（相手の強みを見ることで相手の自信が増す）
  • きっかけを与えて決断を促す（具体的な事例や分かりやすい話で「なるほど」と思わせる）
  • 相手の負担を軽くする
  • 見通しを伝える（時間軸で将来像を示すことで相手の安心感を引き出す）

信頼関係を築くコミュニケーション

← S03:p.54-56

① 相手の話をよく聞く

  • 自分の知らない、相手が特別に抱えている事情に気づける
  • 一方的な押しつけや誤解を避けることができる
  • 相手が大事にしていること、好みや関心がわかれば重点を置くポイントがわかる

② 相手の状態に合わせる（ミラーリング）

KEY TERM — ミラーリング（Mirroring）

相手の声のトーン・口調・言葉遣い・姿勢などをさりげなく真似て、相手を鏡に映すように意識的に状態を合わせること。話すリズム・スピード・呼吸まで合わせることで相手は安心する。

← S03:p.55

③ 相手の期待を上回ること

KEY POINT

信頼関係構築において最も重要。約束した納期より早く、約束した数量より多く、約束したクオリティより高いものを届けること。「この人と付き合うことで利益があり、快適である」と相手に思わせる。

← S03:p.56
🎙 AUDIO SUPPLEMENT

快適なものは誰しも好む。相手の期待を上回ることで「約束を守るし確実にやる、それどころかそれ以上の結果を出せる」という信頼が積み重なり、信頼関係を築くコミュニケーションの上で最も大切なことになる。

← S05

相手を動きやすくするコミュニケーション

← S03:p.57-60
工夫内容ポイント
① 小さなことから巻き込む 最初から重要・難しいことに巻き込まず、時間もお金もかからない負担の少ないことを頼む 一緒に作業した経験により一体感・仲間意識が生まれ、より難しいことへも挑戦できるようになる
② 対比の法則を使う（二者択一） 「AとBどちらがよいですか？」と二者択一にする 選択の負担が減り、比較的楽な方を選ぼうとする。ただし本来は第3案もあることを意識しておく。
③ 具体的な報告がしやすい情報を提供する 背景・経緯・具体的情報をわかりやすくまとめて渡す 誰しも自分の行動を報告する上司がいる。箇条書き1枚にまとめることで「報告しやすい」と感じ動いてくれる
④ 権威のある他者の言葉を引用する メディア・著名人・信頼できる人物の承認・推薦を使う 人は目の前の相手よりも他者の言葉を信用する傾向がある（口コミ効果）
🎙 AUDIO SUPPLEMENT

対比の法則（二者択一）について：世の中全ては二者択一ではない。A案・B案より優れたC案（第三案）がある可能性を常に意識しておくことが大切。

← S05

第4章まとめ ← S03:p.61

  • 人を動かすためには、人間関係を築き、相手に理解・納得させ、そして行動を促す必要がある
  • 信頼関係を築くには、相手の状態に合わせながらまずは話を聞き、常に相手の期待を上回ることが重要
  • 相手を動きやすくするには、より負担の低いことに巻き込み、権威のある他者からの承認や推薦があることを伝える

第4回 全体まとめ ← S03:p.62

  • ビジネスコミュニケーションを成功させるためには、相手への「心遣い」「気働き」が必要である
  • 傾聴は相手の自発性を大切にしたコミュニケーションの手法である
  • アサーションとはより良い人間関係を築くためのコミュニケーションツールである
  • 人を動かすためには、信頼関係を築き、相手の負担を減らす様々な工夫を施す必要がある

⭐ 補足A: 最重要ワード一覧

用語読み定義・説明重要度
ビジネスコミュニケーション ― ビジネス場面で使う、論理的なメッセージで相手を説得・納得させ、動かすためのコミュニケーション ★★★
傾聴 けいちょう 注意を払い、共感的な態度で真摯に相手の話に耳を傾けるコミュニケーション技法。「積極的傾聴（Active Listening）」とも呼ばれる（ロジャーズ提唱） ★★★

📚 補足B: 知っておくべき背景知識

傾聴とロジャーズの3原則

[外部補足・確認済: 厚生労働省こころの耳・複数心理学資料]
傾聴（積極的傾聴 / Active Listening）の概念は、アメリカの臨床心理学者カール・ロジャーズ（Carl Rogers, 1902–1987）が来談者中心療法において体系化。有効な傾聴のための3原則は以下の通り：

① 共感的理解（Empathic Understanding）
相手の立場に立って、気持ちに共感しながら理解しようとすること ② 無条件の肯定的関心（Unconditional Positive Regard）
善悪・好き嫌いの評価を入れずに聴く。相手の話を否定せず、背景に肯定的な関心を持つこと ③ 自己一致（Congruence）
聴き手が相手にも自分にも真摯な態度で、話が分かりにくい時は真意を確認すること

この3原則を満たすことで、話し手の自発的な成長・変容が促されるとされる。現代では企業内コーチング・1on1・カスタマーサービスまで広く応用されている。

アサーション・トレーニングの歴史

[外部補足・確認済: 複数心理学・HR資料]
アサーショントレーニングは1950年代にジョセフ・ウォルピ（Joseph Wolpe）が行動療法として開発。1970年代にアメリカの公民権運動・ウーマン・リブ運動の文脈で「誰もが自己表現する権利を持つ」という人権思想と結びつき広く普及した。
日本には1990年代に心理療法家の平木典子（国際基督教大学名誉教授）によって紹介・体系化され、企業研修・ハラスメント防止教育などで広く活用されている。

DESC法の提唱者

[外部補足・確認済: HR・心理学資料]
DESC法はアメリカの心理学者ゴードン・バウアー（Gordon H. Bower）が提唱した自己主張の4段階フレームワーク。アサーションスキルを体系的に実践するためのツールとして、現在もビジネス研修や職場のコミュニケーション改善プログラムで広く採用されている。

ミラーリング（同調行動）の心理学的背景

[外部補足・確認済: 複数コミュニケーション心理学資料]
ミラーリングは「カメレオン効果」とも呼ばれる。人は無意識に相手の姿勢・表情・声のトーン・言葉遣いを模倣する傾向があり、意識的に行うことで安心感・親近感・信頼感を高める効果がある。「ペーシング」（呼吸・話速を合わせる）とセットで用いると、さらに効果的とされる。

権威の引用（社会的証明・権威原理）

[外部補足・確認済: 一般的心理学知識]
心理学者ロバート・チャルディーニ（Robert B. Cialdini）の「影響力の武器」（1984年）において、「権威（Authority）」は人を動かす6原則の一つとして位置づけられている。残り5つは「返報性」「コミットメントと一貫性」「社会的証明」「好意」「希少性」。専門家・著名人・メディアの言葉が説得力を持つのはこの権威原理による。

🔗 補足D: 関連概念・次回への接続

ノンバーバルコミュニケーション（非言語コミュニケーション）

[外部補足]
本講義で言及された「ジェスチャー・声のトーン・表情・アイコンタクト」はすべて非言語コミュニケーション（Non-verbal Communication）の要素。アルバート・メラビアン（Albert Mehrabian）の研究では、感情・態度の伝達において言語の内容よりも声のトーンや表情などの非言語要素が大きな影響を与えることが示されている（「メラビアンの法則」）。ただしこの法則は感情コミュニケーションに限定した結果であり、すべての状況に適用できるわけではない点に注意。

アサーションの4つの柱

[外部補足・確認済: アサーション研修資料]
アサーティブなコミュニケーションを実践するための4つの柱として「誠実」「率直」「対等」「自己責任」が挙げられる。誠実・率直・対等は「相手に向き合おうとする自分の気持ち」、自己責任は「自分が言ったこと（あるいは言わなかったこと）に対する責任を自分で引き受けること」を指す。

EQ（感情的知性）とビジネスコミュニケーション

[外部補足]
本講義で扱う「相手の感情を読む」「自分の感情をコントロールする」「共感的理解」はEQ（Emotional Intelligence / 感情的知性）の中核能力と重なる。ダニエル・ゴールマン（Daniel Goleman）の研究では、EQがリーダーシップ能力・ビジネス成果と強く相関することが示されている。

積極的傾聴（Active Listening）と1on1ミーティング

[外部補足]
傾聴はロジャーズの提唱した「積極的傾聴（Active Listening）」とも呼ばれ、単に「聞く」だけでなく能動的に相手の話を理解しようとするコミュニケーション姿勢を指す。現代の職場での1on1ミーティング（上司・部下の定期面談）において中核的スキルとして重視されており、特にコーチング型1on1は人材育成の場として広く実践されている。

影響力の武器（チャルディーニ）との対応関係

[外部補足]
第4章で扱った技法はチャルディーニの「影響力の武器」（Influence: The Psychology of Persuasion, 1984）の概念と対応している：
・「権威の引用」→ 権威（Authority）
・「小さなことから巻き込む」→ コミットメントと一貫性（Commitment & Consistency） ＋ フット・イン・ザ・ドア技法
・「対比の法則（二者択一）」→ コントラスト効果（Contrast Principle）
・「口コミ・評判」→ 社会的証明（Social Proof）
