第1章 社会人基礎力が必要な理由

職場に定着できない若者の増加

厚生労働省「新規学卒就職者の離職状況（令和3年3月卒業者）」によると、若者が就職後3年以内に職場を去るケースが増加しており、社会問題となっている。 S03:p.4

🔊 音声補足
大学卒の3年以内離職率（令和3年3月卒・確定値）
  • 3年以内合計：34.9%（3人に1人以上が3年以内に離職）
  • 1年目：12.3% / 2年目：12.3% / 3年目：10.3%
  • 大卒は他学歴と比べて1年目の早期離職が少なく、2〜3年目にかけて均等に離職する傾向がある
  • 最新（令和4年3月卒）では33.8%とやや低下傾向にあるが、依然として3割超が継続
なぜ離職率が高まったかという点を、社会人基礎力との関係で説明している。 ← 厚生労働省公式データ Web照合済み [外部補足]

[図解省略：スライドp.4 就職後3年以内離職率の推移グラフ参照]

教育環境の変化
🔊 音声補足
コミュニケーションが減ったのではなく、形が変化した（携帯・パソコン・インターネット）。しかし対面・家族・地域内のコミュニケーションは随分減っている。大学全入時代については、サイバー大学の学生には当てはまらない部分もあると補足されている。 ← S05:ch1
ビジネス環境の変化
🔊 音声補足
「いいものを作れば売れる」時代から、個々のニーズに沿った商品・サービスが求められる時代へ。最小限の人員で最高の効果を出す（スリム化）中で、一人ひとりが考える領域が増えている。過去の常識にとらわれない発想が求められるようになった。 ← S05:ch1
社会人基礎力を構成する3つの能力（復習）

経済産業省が2006年に提唱した「社会人基礎力」は、職場や地域社会で多様な人と仕事をしていくために必要な基礎的な力を3つの能力・12の能力要素として定義している。S03:p.7,9

「人生100年時代」や「第四次産業革命」の下で、2006年に発表した社会人基礎力はむしろその重要性を増している。S03:p.7

🔊 音声補足
今後のキャリアが多様化・社会が複雑化し、曖昧な価値観への対応が難しくなっている中でも、社会人基礎力をしっかり抑えておけば対応できる、と説明されている。 ← S05:ch1

第1章まとめ：若者が大学を卒業するまでに身につける能力と、職場で求められる能力が合わなくなってきた。だから早期に社会人基礎力を身につけることが重要。S03:p.13

第2章 前に踏み出す力（アクション）

前に踏み出す力は以下の3要素で構成される。S03:p.10,17

正確な3要素：主体性（物事に進んで取り組む力）・働きかけ力（他人に働きかけ巻き込む力）・実行力（目的を設定し確実に行動する力）S03:p.17

指示待ちにならず、一人称で物事を捉え、自ら行動できるようになることが求められている。S03:p.10

① 主体性

【定義】 物事に進んで取り組む力 S03:p.18

✗
仕事が終わったので、上司が次の指示を出してくれるのを待つ。
○
自分でやるべきことを見つけ、上司に相談して進める。

← S03:p.18

🔊 音声補足（インターンシップへの適用）
頼まれた1つの仕事が終わったら、上司や受け入れ担当者が声をかけてくれるのを待つのではなく、①終わったことをきちんと報告し（報告・連絡・相談）、②自分から次の指示をもらいに行く（「次に私は何をすればいいですか」と聞く）。また、自分ができること（直接業務に影響しない作業等）を見つけ、相談しながら仕事をもらう。 ← S05:ch2
🌐 外部補足
評価される理由は2つ：①言われた以上のことをする姿勢が評価される、②自分で考えることで一つひとつの仕事の意味が理解でき、自己成長につながる。 ← S05:ch2
② 働きかけ力

【定義】 他人に働きかけ巻き込む力 S03:p.20

✗
「会議室の机がいつも汚れている」と文句を言いながら拭いている。
○
「会議終了後に毎回机を拭くようにしよう」と自分が率先し、みんなに声をかける。

← S03:p.20

🔊 音声補足（インターンシップへの適用）
他のインターンシップ生や社員のモチベーションが低い場合、自分もやる気をなくすのではなく、周りのモチベーションを上げる方法を考える。やる気がない原因を取り除く手助けをし、「一緒に頑張ろう」と声をかける。まず自分が主体的に動き、その後に声をかけながら盛り上げる。 ← S05:ch2
🔊 音声補足
「人間の行動には確実にその裏側にその行動を起こした原因がある」。やる気のない社員が「仕事はそんなに楽しいものじゃない」と言っても、インターンシップ生の積極的な姿勢がその社員の意識を変えるきっかけになる可能性がある。 ← S05:ch2
③ 実行力

【定義】 目的を設定し確実に行動する力 S03:p.22

✗
取引先に新サービスを案内するよう言われたので、定期訪問の際に担当者にチラシを渡す。（単に渡すだけで相手に真意が伝わらない）
○
「新サービスの契約を10件獲得する」という目標を立て、いくつかの手段を考えて実行する。

← S03:p.22

🔊 音声補足（インターンシップへの適用）
名刺のデータ入力のような「つまらない仕事」でも、「今週中にこれだけの数の入力を終える」「この会社の取引先企業名を〇件覚える」などの目標を自分で設定し、達成まで粘り強く取り組む。つまらない仕事でも目標設定があれば「楽しい仕事・身につく仕事」として取り組めるようになる。小さなことをきちんとやるインターンシップ生には、次の課題としてやりたいことが任せられるようになる。 ← S05:ch2
🔊 音声補足
実行力のある社員・職員は常に自分たちの行動を何かの成果につなげながら行動している。 ← S05:ch2
第2章まとめ

「やらされている」という感覚は捨て、すべてのことは「自分の仕事」という意識で取り組む

S03:p.24
🔊 音声補足
自分の目の前にある仕事はだれの仕事でもなく、自分自身に任せられた仕事。どのような意識で取り組むかによって、その仕事を通じて学べる量・出せる成果は大きく異なる。 ← S05:ch2

第3章 考え抜く力（シンキング）

考え抜く力は以下の3要素で構成される。S03:p.28

論理的に答えを出すこと以上に、自ら課題を定義し、解決のためのシナリオを描く自律的な思考力が求められている。S03:p.11

① 課題発見力

【定義】 現状を分析し目的や課題を明らかにする力 S03:p.29

🔊 音声補足（作業と仕事の違い）
「作業」とは自分であまり考えなくてよい、とりあえず言われたことをやればよいもの。「仕事」とはその中で考えていかなければならないもの。「作業」を「仕事」に変えるためには課題を発見する視点が必要。 ← S05:ch3
✗
「この作業は時間がかかるから嫌だ」と思いながら仕事をしている。（仕事の意味を考える視点がなくなる）
○
作業を効率的に進められない問題を見つけ、解決することを周囲に提案する。

← S03:p.29

🌐 外部補足（三木谷浩史社長の言葉）
楽天グループの三木谷浩史社長は著書『成功のコンセプト』（幻冬舎、2007年）の中で「1日に1%の改善を続けると1年間（365日）で約37倍の効率を得ることができる」と述べている。計算式：1.01365 ≈ 37.78。「常に改善、常に前進」を第1のコンセプトとして位置づけている。 ← S05-A・書籍確認済み ※S03に記載なし
🔊 音声補足（インターンシップへの適用）
職場に書類が多く、みんなが探すのに苦労していることに気づいた場合、「この会社のやり方だから」と思うのではなく、なぜ探しにくいのかという原因を考える。ただし勝手に整理するのではなく「使いやすいように整理しましょうか？」と相談する（書類の配置は各自が自分の使いやすいように置いているため）。 ← S05:ch3
🔊 音声補足（整理と整頓の違い）
  • 整理：不要なものを捨てること（要る/要らないの判断は社員に確認する。機密書類は重要廃棄物として適切に処理する）
  • 整頓：必要なものを分かりやすく配置すること（ファイル整理・インデックス付与など、検索性が高い状態にする）
コミュニケーションなしで勝手に行うより、相談しながら進める方が組織にとって有益。 ← S05:ch3
② 計画力

【定義】 課題の解決に向けたプロセスを明らかにし準備する力 S03:p.31

✗
課題を解決するために何をすればよいか全くわからず、手がつけられない。（課題を溜め込むだけ、「時間がない」と言い訳）
○
課題を解決する方法を数パターン考えて頭の中でシミュレーションし、最善策を検討。実行の準備を進める。

← S03:p.31

🔊 音声補足（インターンシップへの適用）
職場の書類を利用しやすいように整理することになった場合、「順番に並べればよい」と安易に考えず、最も利用しやすい保管方法は何かを複数案検討する。「誰が利用しやすいのか」を考え、様々な人の意見を聞く。意見が増えて「聞けば聞くほど分からなくなる」ときは、自分で複数の解決策を作り上げて、周囲の社員に分かりやすく説明し選んでもらう。 ← S05:ch3
🔊 音声補足
仕事は「自分だけで回答を探す作業」ではなく、「周囲と話し合いながら最善のものを探していくプロセス」。周囲との関係性の中で納得解を探し、みんなが納得できる形を計画・実行することが重要。 ← S05:ch3
🔊 音声補足
「時間がない」と言う人は実際には解決するプロセスを考えていないのではないか、という指摘がある。時間は作ろうと思えば作れる。 ← S05:ch3
③ 創造力

【定義】 新しい価値を生み出す力 S03:p.33

✗
今までのやり方、自分の知っている範囲で課題を解決しようとする。
○
様々な情報を収集し、課題を解決するための新しいアイデアを考え出す。

← S03:p.33

🔊 音声補足（インターンシップへの適用）
書類の整理方法を考える際、自分の知識の範囲内だけで考えるのではなく：利用者の意見を聞く、世の中の分類方法を調べる、データ化する方法を考える（紙でなくサーバー・パソコンで管理）など、広い視野で解決方法を探す。書類の整理だけでも、それを使っている人→その人が行う仕事→その先のお客様まで、頭の中がどんどん活性化していく。 ← S05:ch3
🔊 音声補足
「楽しい」と「楽」は違う。楽になった分の時間で新しいアイデアを出していくことが、仕事を本当に楽しくすることにつながる。前回よりも今回、今回よりも次回、という改善の姿勢が重要。 ← S05:ch3

第3章まとめ：すぐに答えを求めようとせず、自分で調べる・考えるという手順を踏む

S03:p.35

第4章 チームで働く力（チームワーク）

チームで働く力の6要素を、以下の3グループに分類する。S03:p.39

グループ内の協調性だけに留まらず、多様な人々との繋がりや協働を生み出す力が求められている。

① コミュニケーションに必要な力

【定義（3つの要素）】S03:p.40

  • 発信力：自分の意見をわかりやすく伝える力
  • 傾聴力：相手の意見を丁寧に聞く力
  • 状況把握力：自分と周囲の人々や物事との関係性を理解する力
✗
会議などで一方的に自分の意見だけを主張し、周囲の意見を言わせない・認めない。（これは発信力ではない。相手に押し付けになっているため）
○
相手が理解しやすい言葉で話し、適切なタイミングで質問し、周囲の意見も引き出す努力をする。

← S03:p.40

🔊 音声補足（会議の目的）
会議は「合意（合議）」のためにある。参加している一人ひとりがそこで合意を形成し、自分が会議終了後に何をすべきかを理解し、その後の行動につながるのが本当の会議。これが理解できていなければ状況把握力が欠如している。 ← S05:ch4
🔊 音声補足（インターンシップへの適用・メモを取る意味）
受け入れ企業の担当者から作業の指示を受けたとき、言われたことだけを聞くのではなく、担当者の話をメモを取りながら聞き、その作業の意味を理解するために質問し、業務全体の把握に努める。メモを取る意味は2つ：①言われたことを忘れない、②相手に「一生懸命聞いている」という姿勢を伝える。メモを取ることで相手がもっと教えてくれるようになり、質問への回答も丁寧になる。 ← S05:ch4
② ストレスに対応する力

【定義（2つの要素）】S03:p.42

  • 柔軟性：意見の違いや立場の違いを理解する力
  • ストレスコントロール力：ストレスの発生源に対応する力
🔊 音声補足（離職との関係）
離職した方へのインタビューで「職場の人間関係に疲れた」という意見が非常に多い。職場の人間関係は「自分で作るもの」。その作り方がうまくなかった、または知らなかったことでストレスを溜めてしまい職場を離れる。だからストレスに対応する力は事前に学習する必要がある。 ← S05:ch4
✗
上司と意見が合わず、職場での人間関係にストレスを感じている。
○
上司の立場で考え、なぜそのような意見であるかを理解し、会社にとって何が最善かをその意見も含めて一緒に考える。

← S03:p.42

🔊 音声補足（インターンシップへの適用）
受け入れ企業の担当者から態度について注意を受けた場合、「厳しすぎる」と思うのではなく、「注意してくれる＝もっとよい形にしたいという気持ちの裏返し」と捉える。インターンシップ生の態度は対外的には受け入れ企業の評価になるため、企業の立場を考えて改善する。注意されたことで「この企業はどういう姿になりたいのか」を理解するヒントにもなる。そう考えればストレスはたまらない。 ← S05:ch4
③ 自分を律する力（規律性）

【定義】 社会のルールや人との約束を守る力 S03:p.44

✗
お客様との約束の時間に連絡なしで遅れる。（相手が待ち続けることになる）
○
お客様との約束の時間は厳守する。事故などやむを得ない場合はすぐ連絡してお詫びする。

← S03:p.44

🔊 音声補足（インターンシップへの適用）
作業にソフトウェアのインストールが必要になった場合、企業のPCに勝手にインストールするのではなく、受け入れ企業の担当者に「何のために何のソフトのインストールが必要なのか」をきちんと説明・相談する。企業には厳格な規約がありインストールできるソフトウェアが決まっている場合もある。その規律が何のために存在するのかを理解した上で行動する。 ← S05:ch4
🔊 音声補足
ルールは「周囲も自分も気持ちよく仕事ができるように」決まっているもの。相手の立場に立った言動を常に考えれば、そのルール・マナーを常に意識できるようになる。 ← S05:ch4

第4章まとめ：ビジネスマナー・社会のルールを守ることは当然。相手の立場に立った言動を心がける

社会人基礎力の経緯（確定）

経済産業省が主催した有識者会議により2006年に提唱（経済産業省公式サイトで確認済み）。職場や地域社会で多様な人と仕事をしていくために必要な基礎的な力を「3つの能力・12の能力要素」として定義した。 その後、2018年に「人生100年時代の社会人基礎力」として再定義。「目的（どう活躍するか）」「学び（何を学ぶか）」「統合（どのように学ぶか）」の3つの視点が追加され、新入社員だけでなく全世代を対象とした考え方に拡張された。 詳細は経済産業省ウェブサイト（https://www.meti.go.jp/policy/kisoryoku/index.html）で確認できる。 ← S03:p.7,8・Web照合済み [外部補足]

厚生労働省の離職率データ（確定数値）

スライドで引用されているのは「厚生労働省 新規学卒就職者の離職状況（令和3年3月卒業者）」のグラフ。Web照合により数値が確定した。 令和3年3月卒・大学卒の3年以内離職率：34.9%（前年比 +2.6ポイント）。内訳は1年目12.3%・2年目12.3%・3年目10.3%で、大卒は他学歴と比べ1年目の早期離職が少ない特徴がある。 最新データ（令和4年3月卒）では33.8%とやや改善しているが、依然として3割超の水準が続いている。 出典: 厚生労働省「新規学卒就職者の離職状況」（https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000137940.html） ← S03:p.4・Web照合済み [外部補足]

1日1%改善 → 1年で37.8倍 の出典（確定）

三木谷浩史（楽天グループ創業者）著『成功のコンセプト』（幻冬舎、2007年）に記載された考え方。 計算式：1.01365 = 37.78倍。「毎日1%の改善を1年続ければ37倍の効率を得られる」という言葉は、同書の「常に改善、常に前進」という第1のコンセプトとして紹介されている。 授業中は音声のみで言及（S03に記載なし）。数式での説明は本書を参照のこと。 ← S05-A・出典確認済み [外部補足]

🔗 補足D: 関連概念・次回への接続

【外部補足】「人生100年時代の社会人基礎力」（2018年再定義）：2006年版の3能力・12要素を土台に、2018年に経済産業省が再定義。新たに「目的（どう活躍するか）」「学び（何を学ぶか）」「統合（どのように学ぶか）」の3つの視点が追加された。対象が大学生・新入社員だけでなく、就学前からシニア層まで全世代に拡張された点が特徴。自己のキャリアを自らが切り開く「キャリアオーナーシップ」の意識も重要とされている。出典：経済産業省 https://www.meti.go.jp/policy/kisoryoku/index.html

【外部補足】報告・連絡・相談（ほうれんそう）：授業内で「前回も話した」と言及されている。仕事の完了を上司に報告し、次の指示を求める基本的なビジネスコミュニケーション。主体性の発揮と直結している。

【外部補足】VUCA（Volatile, Uncertain, Complex, Ambiguous）：現代社会の複雑性・曖昧性を表す概念。「キャリアの多様化・社会の複雑化・曖昧な価値観」という授業の文脈と接続する。社会人基礎力はこのVUCA時代に対応するための基礎となる。

【外部補足】PDCAサイクル（Plan-Do-Check-Action）：計画力・課題発見力・創造力の繰り返し改善の考え方と関連する。三木谷浩史著『成功のコンセプト』の「1日1%の改善」というエピソードはPDCAの継続的改善（Kaizen）の思想とも一致する。
