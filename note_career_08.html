第1章

本科目の振り返り

1-1. 学習目標の確認 ← S03:p.4

#学習目標
1次世代における企業や組織が求める人材像を理解する
2次世代における企業や組織で働くために必要な能力を理解する
3ビジネスをプロデュースする上で必要な能力を理解する
超情報化時代においては「知識・情報を持つ」こと自体は当たり前になっている。そこから何を創造し、何を考えられるかが問われる。

1-2. 企業が求める次世代人材の背景 ← S03:p.5

① 少子高齢化の影響

  • 働き手の減少
  • マーケット自体の少子高齢化 → 従来の商品・売り方が通用しなくなる

② 消費者メディアの台頭による市場変化

  • 人と人がつながり口コミが生まれる → 価値の多様化・ニーズの多様化
  • 消費者の意思決定プロセス・行動の変化

③ 3つのパラダイムシフト → これからの企業の課題

  • 多様な人材が協働できる組織への変革
  • 海外市場開拓に必要なリソースの確保
  • 社外パートナー・顧客を巻き込んだ新しいビジネスの創造
  • ソーシャルメディアを活用した顧客との新たな関係構築
結論：ITによる「効率化」から、ITを活用した「変革（transformation）」へ。変革とは「今までの価値観を捨て、新しい価値観を作ること」。
→ 「効率」ではなく「変革」を重視したビジネスプロデュース力が求められる。 ← S03:p.5

1-3. 企業や組織で働くために必要な能力（3層構造） ← S03:p.6

層能力内容
基盤社会人基礎力一歩踏み出す・深く考える・仲間と協働
中間プロジェクト遂行力プロジェクトを前に進める力・メンバーとしての考え方
上位ビジネスプロデュース力既存価値を再編集・リニューアルして蘇らせるリノベーション的発想
「ビジネスプロデュース力」とは0から1で作ることではない。既存の価値観を合わせ、リニューアルして、場を変えて蘇らせる——リノベーションに近いものが求められている。

1-4. ビジネスプロデューサーに必要な7つの能力 ← S03:p.7

出典：吉田就彦著『アイデアをカタチにする仕事術』東洋経済新報社 p.57

能力定義
発見力チャンスやヒットの芽や新しい人財などを発見する力
理解力世の中の動きや物事の本質を理解する力
目標力ビジョンを描きゴールをイメージできる力
組織力さまざまなビジネス資源を組織して有効活用する力
働きかけ力「人」や組織を励まし、力を吹き込み、目標に向かって育てる力
柔軟力トラブルや環境変化に対応するなど、柔軟に物事を調整する力
完結力さまざまなことを乗り越えて実行し、達成して、次への蓄積とする力

1-5. 既存の枠にとらわれず、自由に発想する ← S03:p.8

① 思考のクセ

  • 過去の経験・知識に影響され、連鎖的思考が幅広い視点での発想を阻害する
  • 新しい時代においては過去の経験・知識が全て「昔の話」になる可能性がある

② 常識のカベ

  • 社会的・組織的・個人が持つ常識や規範が斬新な発想を阻害し否定させる
  • 常識によって「現状に対する疑問」が失われる
  • 「最近おかしいな」という直感は正常な反応——常識のカベが強いと自分がおかしいと感じてしまう
克服法：視点の偏りを認識して、意識的に視点を広げる・ずらす ← S03:p.8

1-6. 第1章まとめ：今、何に取り組むか ← S03:p.9–10

行動・成果 = 意欲 × 能力
  • 能力の差：「ある人」と「ない人」は一般的に2倍程度
  • 意欲の差：1,000倍〜10,000倍（掛け算なので意欲が最大の変数）
  • 能力に偏りすぎると逆に意欲を下げてしまうことがある
結果よりも行動にフォーカスする。インターネット・ソーシャルメディアを駆使し他者の能力を借りることも有効。どれだけの行動量を確保できるかを意識して進めることが重要。
第1章の結論：次世代に必要とされる人材像を明確にし、今の自分に足りないスキルを把握する。そのギャップを埋めるために今から何に取り組み、どのような行動をとるか、計画を立てる。 ← S03:p.10
第2章

「キャリア」を考える上で大切なこと

2-1. モチベーションとキャリアの違い ← S03:p.14

概念時間軸内容
モチベーション短期半年のプロジェクト、3ヶ月の資格取得など
キャリア長期（定年まで）自分にとって意味のあることを成し遂げていく積み重ね
キャリアを考えることは、組織でいう"戦略"を立てるようなもの。自分の人生をいかにより良いものにしていくか——その設計がキャリア設計。 ← S03:p.14

2-2. 変化が激しい時代のキャリアデザイン ← S03:p.15

  • 市場予測が非常に難しい時代（顧客が何を欲しているか自体が予測困難）
  • 戦後高度経済成長期の価値観・戦略では通用しない
  • 会社は方向性を示しながら各人の成長をサポートする使命を持つ
  • 20代→40代→60代と、自分を取り巻く時代環境も変化していく
環境が激しく変化する現代では、各人が主体的にキャリアをデザインする必要がある。
プロセス：予測する → イメージする → 自分をマッピングする → 今日の行動を逆算で作る ← S03:p.15

2-3. キャリアデザインの不確実性 ← S03:p.16

キャリアの将来像が「予期せず崩れてしまう」3つの要因：

  1. ビジネス環境の変化
  2. 生活環境の変化
  3. 会社に起こり得る変化（方針変更など）
「安定した会社」「安定した職業」は現時点では安定しているように見えるに過ぎない。潰れない会社はあり得ない。安定した職業もない。過去の流れでそう見えた会社・職業が、予期せず崩れたとき、そこにいる人たちは全員影響を受ける。

2-4. 変化に順応する柔軟性 ← S03:p.17

自己イメージの実現を目指しながら、職業人生の中で起こる変化に柔軟に対処できるよう準備しておくことが必要。

変化の例対応
配置転換（望む部署でない配属）調整・修正・自身の解釈を加えながら、自身が描く将来像に近づける
出産・育児（家族構成の変化）
親族の介護（超高齢化社会への対応）

2-5. キャリア開発とは ← S03:p.18

会社・職場で必要とされる人材であり続けるために、不足している知識やスキルを自ら身につけ、仕事を通じて成長していく。
  • 「安定した職場・職業」ではなく「安定した能力」を身につけることが本質
  • 自己防衛の手段として：常に会社から必要とされる知識・スキルを身につけておく
  • 仕事を通じた継続的な成長 → 雇用の継続につながる

2-6. 「成長」は3つの軸で考える ← S03:p.19

③ 職位が上がるとは、自分の能力を上げることからメンバーの能力を上げることに仕事がシフトすること。部下の支援に必死になればなるほどマネジメント能力が上がる——子供を持つと親が成長するのと同じ構造。

2-7. 第2章まとめ ← S03:p.20

キャリア = 能力を高めること。

「自分が何をやりたいか」だけでなく、
・どのような人が必要とされているか
・自分の強みを活かして何ができるか
を考えながら、自分はどういうことをまずすべきかを理解することが重要。

そのため、不確定な未来にも柔軟に対応できる力が必要。
第3章

まとめ

3-1. 社会人の学習習慣 ← S03:p.24

エンプロイアビリティ（雇用され続ける能力）を維持するために、学習習慣を持ち続けることが重要。

  • 年齢が上がるほど学習者率は右肩下がりだが、60〜65歳でも継続的に学習している人は多い
  • 「学んでいるつもり」の落とし穴：情報を大量に見ているだけで、本当の学びになっていないことがある

出典：総務省「令和3年社会生活基本調査」 https://www.stat.go.jp/data/shakai/2021/pdf/gaiyoua.pdf [2024.9.19閲覧]

3-2. 能力を高める努力は自分で続ける ← S03:p.25

スキルは「自分で学び積み重ねていく」もの。既にあるスキルの棚卸しと、陳腐化していないか検証することも重要。

3-3. 求められる人材であり続けるためには ← S03:p.26

3-4. PEST分析で「世の中を知る」 ← S03:p.27

観点内容
Political（政治的）新しい法律の施行、規制強化・緩和、税制、政府の施策・動き など
Economical（経済的）景気の変動、物価の変動、経済指標の変動、株価・金利・為替の変動 など
Social（社会的）人口動態、世論、流行・社会現象、教育水準、自然環境の変化 など
Technological（技術的）技術革新、新しい発明・発見、特許 など
PEST分析は自分のキャリアに影響するマクロ環境を把握するためのフレームワーク。例えばS（社会）の人口動態（少子高齢化）を読めば、どの業界のビジネスが伸び縮みするかを予測できる。そこからP（政治）の規制変化とT（技術）のAI活用を組み合わせて、自分が身につけるべきスキルを逆算することができる。

3-5. 3C分析で「会社の状況を知る」 ← S03:p.28

C視点内容
Customer市場・顧客顧客のニーズや市場の動向
Competitor競合競合他社の動き・強み・弱み
Company自社自社の強み・弱み・資源

3C分析を通じて、自社が将来も有望か・現状維持か・衰退傾向かを予測しながら自分の行動を決める。

3-6. 「自分を知る」 ← S03:p.29–30

側面把握すべき内容
強み性格・身体・対人面・知識量・生活態度・社会性
弱み同上の各側面における課題・改善余地

問い：「伸ばしたいものは何か？」「克服したいものは何か？」

自身の強み・弱みを踏まえた職業選択・働く場の選択が重要。仕事（プロジェクト）の中で自分の役割を確実に、最高品質でこなすことが成長の最短経路。 ← S03:p.30
第4章

次世代人材として活躍するあなたへ

4-1. 次世代人材の定義 ← S03:p.34

  • これまでの世代の延長線上にない
  • 単純に過去の能力を伸ばすだけでなく、新しい時代で必要とされる能力を自ら身につける
  • 求められる能力は常に変化し続けている
企業サイドも「どういう人材にどの能力を持ってもらって」というところが伝えにくくなってきている。それほど時代が大きく変化している。だからこそ、「今までになかった力・今までになかった価値を作り出す力」が安定した能力になる。

4-2. 求められる能力の変化（1999年→現在） ← S03:p.34

1999年に企業が重視した能力（行動力・実行力・熱意・意欲・論理的思考力・創造力・専門知識・研究内容）は「意欲を持って行動する人間」が中心だった。現在のキャリア入門が加えるもの：

  • ビジネスプロデュース力（既存の枠を超えた新ビジネス創造）
  • プロジェクト推進能力（プロジェクト型の仕事の増加に対応）
  • 社会人基礎力（1999年以降に提唱）

出典：経済同友会 教育委員会「企業の採用と教育に関するアンケート調査」結果（2016年調査） https://www.doyukai.or.jp/policyproposals/articles/2016/pdf/161221b.pdf [2024.3.18閲覧]

4-3. 企業の生き残りが難しい時代 ← S03:p.35

  • 大企業の大規模リストラ・相次ぐ大型買収・事業撤退
  • リアル経済圏に加え、EC・ソーシャルメディアによるネット経済圏の台頭
  • 企業が生き残りをかけて最初に確保しようとするもの：「新しい発想ができる人材」
新しいマーケットを創造したり新しいビジネスを作れる人材は全体の1〜2割。その1〜2割は学力では測れない。常に時代とともに変化し、自分自身も変化させる習慣を持ち、常に学習する人材。 ← S03:p.35

4-4. キャリア = 能力を高めること（最終まとめ） ← S03:p.36

「学ぶ力」がある人の4つの習慣：

習慣内容
自分で学ぶ主体的な学習
他人から学ぶ他者の経験・知識から吸収する
環境から学ぶ身の回りの事象から学ぶ
学ぶ環境をつくり出すビジネスプロデュースに近い発想。自分だけでなく周りを動かす

「学ぶ力」に必要な条件：知力・集中力・精神力・繊細さ

参考：苗木邦王著『負け癖社員はクビにしろ！』幻冬舎（P104〜107）

人と相対評価で考える必要はない。昨日の自分より今日の自分が成長したかという指標で十分。 ← S03:p.36

4-5. 時代の変化を楽しむ ← S03:p.37

  • スマートデバイス・クラウドサービス・ソーシャルメディアの普及
  • 「4年前の自分と今」はそれほど変わっていない
  • 「今から4年後」は想像もつかないほど変化している（変化速度は加速）
「従来はできなかったことが実現できるようになっている」。「無理だよ」という常識と戦うと、その"ものさし"を冷静に見ると実はできることに気づく。そのためのツールが次々と新しく出てきているのが現代社会。 ← S03:p.37

4-6. 講師からのメッセージ（最後に） ← S05

講師自身のキャリア軌跡：あるメーカーに就職→11年勤務→独立・会社設立。約10年スパンでいろいろなことにチャレンジし続けた。現在は客員教授・会社2社・NPO3団体・一般社団法人1団体・複数大学での教鞭を並行。

「その時その時に決断できたのは、熱意を持って自分自身が必死にやっていたから。常に高い目標を置きながら日々生活する中で自分のキャリアは積み上げられる。」
次世代は難しい時代でもあるが、「予想以上のキャリア」を作ることができる時代でもある。
自分のキャリアについて深く考え、高い目標を設定し、熱意を持ってチャレンジすること。

B-1. 吉田就彦著書について ✅

『アイデアをカタチにする仕事術――ビジネス・プロデューサーの7つの能力』
著者：吉田就彦（デジタルハリウッド大学大学院専任教授・ヒットコンテンツ研究所代表）

7つの能力（発見力・理解力・目標力・組織力・働きかけ力・柔軟力・完結力）を「能力サイクル」として体系化。「おくりびと」「モンスターハンター」などのヒット事例を分析。

B-2. 社会人基礎力について ✅

経済産業省が2006年に提唱した「職場や地域社会で多様な人々と仕事をしていくために必要な基礎的な力」。「前に踏み出す力（アクション）」「考え抜く力（シンキング）」「チームで働く力（チームワーク）」の3能力・12要素から構成される。
2018年には「人生100年時代の社会人基礎力」として改訂され、「目的」「学び」「統合」の3つの視点が追加された。自らキャリアを切り開いていく上でのリフレクション（振り返り）が重視されている。
確認ソース：経済産業省公式 meti.go.jp/policy/kisoryoku/index.html

B-3. PEST分析の起源 ✅

ハーバード・ビジネス・スクール教授のFrancis J. Aguilar（1932–2013）が1967年の著書 Scanning the Business Environment で提唱。元の名称はETPS（Economic, Technical, Political, Social）で、その後PEST・PESTLEなどへ変化した。現在は世界中でマクロ環境分析の基礎フレームワークとして使われている。
確認ソース：Wikipedia英語版 en.wikipedia.org/wiki/Francis_J._Aguilar ・ Wikipedia PEST分析

B-4. 3C分析の起源 ✅

経営コンサルタント（元マッキンゼー日本支社長）の大前研一が1982年の著書 The Mind of the Strategist: The Art of Japanese Business（邦訳：『ストラテジック・マインド』）で提唱。市場・顧客（Customer）・競合（Competitor）・自社（Company/Corporation）の3者の関係を「戦略的三角関係（Strategic Triangle）」と呼ぶ。
PEST分析が「自社でコントロールできないマクロ環境」を分析するのに対し、3C分析は「ある程度影響力を行使できるミクロ環境」の分析に用いる点で補完的な関係にある。
確認ソース：ITメディア情報システム用語事典 itmedia.co.jp/im/articles/0705/30/news141.html ・ 複数のマーケティングサイトで一致

B-5. エンプロイアビリティとは

終身雇用が崩壊しつつある現代において、特定の会社・職業の「安定性」に依存するのではなく、「どこでも雇用され続けられる能力・市場価値」を維持し続けるという概念。変化への継続的な学習・適応能力が核心となる。本講義では社会人の学習習慣の維持に直結する概念として取り上げられている。

B-6. パラダイムシフトとは

科学哲学者Thomas Kuhn（1962年『科学革命の構造』）が提唱した概念。単なる「改善」ではなく、前提となる考え方自体が置き換わるほどの変化を指す。本講義では情報化社会・グローバル化・少子高齢化による社会・経済の構造的変化という文脈で使われている。

付録C：試験対策ポイント
  1. 行動・成果 = 意欲 × 能力の式の意味——能力差は2倍程度、意欲差は1,000〜10,000倍
  2. ビジネスプロデューサーに必要な7つの能力の名称と定義（発見力・理解力・目標力・組織力・働きかけ力・柔軟力・完結力）。出典：吉田就彦著（東洋経済新報社, 2010）
  3. PEST分析の4視点と各視点で見るべき内容（Political・Economical・Social・Technological）
  4. 3C分析の3要素（Customer・Competitor・Company）
  5. キャリア ≠ モチベーションの違い：時間軸（長期 vs 短期）・目的の違い
  6. 「安定した職場・職業」ではなく「安定した能力」という考え方
  7. 成長の3つの軸：①できることの幅を広げる、②より重要な役割を担う、③職位が上がる
  8. 新しいビジネスを創れる人材は全体の1〜2割で、学力では測れない
  9. 社会人基礎力：経済産業省2006年提唱、3能力12要素、2018年「人生100年時代の社会人基礎力」に改訂
  10. 学ぶ力の4つの習慣：自分で・他人から・環境から・学ぶ環境をつくり出す
付録D：関連概念・接続 （✅=一次ソース確認済）
  • ビジネスプロデュース力 × プロジェクト遂行力 × 社会人基礎力：3つの歯車が噛み合って初めて次世代人材として機能する
  • PEST分析と3C分析の連携：PEST（マクロ環境）で世の中全体を把握し、3C（ミクロ環境）で自社の立ち位置を把握。両者を組み合わせることで「自分が今なすべきこと」が見えてくる
  • 計画的偶発性理論（プランド・ハプンスタンス） ✅
    スタンフォード大学のJohn D. Krumboltz教授が1999年に提唱（Mitchell, Levin, & Krumboltz, 1999, Journal of Counseling & Development, 77(2), 115–124）。「キャリアの8割は予期しない偶発的な出来事によって決まる」とし、偶然をチャンスに変えるために5つの行動特性を推奨：好奇心・持続性・柔軟性・楽観性・冒険心（リスク・テイキング）。本講義の「不確定な未来への柔軟な対応」「主体的なキャリアデザイン」と強く共鳴する。
    確認ソース：Wikipedia日本語版「計画的偶発性理論」・Mitchell et al.(1999)原典DOI確認
  • プロティアン・キャリア（Protean Career）
    ハーバード大学のDouglas T. Hall教授が1976年に提唱したキャリア論。組織ではなく個人がキャリアの主体となり、自分の価値観に従って柔軟に変化し続けるキャリア観。本講義の「主体的なキャリアデザイン」「安定した能力の維持」と一致する現代的なキャリア観。[外部補足]
  • ジョブ・クラフティング（Job Crafting）
    Yale大学のAmy Wrzesniewskiらが2001年に発表した概念。自分の仕事の境界・目的・関係を自ら再構成しやりがいを創出すること。本講義の「学ぶ環境をつくり出す」「ビジネスプロデュース力」という能動的姿勢と親和性が高い。[外部補足]
