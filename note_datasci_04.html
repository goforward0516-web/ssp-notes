第3回の復習

冒頭で確認問題が出された。← S03:p.2-3

📬 Q. ある1週間に届いたメールの件数
曜日月火水木金土日
件数181211101423

1週間のメール件数の平均・標準偏差を求めよ。

解答
平均 x̄ = (18+12+11+10+14+2+3) / 7 = 10（本）
標準偏差 σ = √[(18-10)²+(12-10)²+…+(3-10)²] / 7 ≈ 5.32（本）
🎤 音声補足 ← S05a
どちらも普段のデータでは計算機で計算することが多いが、非常に基本的な概念なので、自分でも簡単なデータの場合は計算できるようにしっかり覚えておくこと。

第1章　複数のデータの扱い

1-1. 前回との繋がり

第2回ではスマホゲームの満足度調査を題材に、年代で層別抽出したユーザーを対象にアンケート調査を行うことを学んだ。第3回では年齢データを例に1種類のデータの全体像を把握する手法（度数分布表・ヒストグラム・平均・中央値・分散・標準偏差）を学んだ。← S03:p.8-9

🎤 音声補足 ← S05a
アンケートデータを取った場合、年齢データだけでなく年齢・職業・プレイ時間・満足度など複数のデータを収集することになる。この複数のデータをどのように比較したらいいのかを考える必要が出てくる。

1-2. 対応のあるデータ

定義対応のあるデータ: 同一の観測単位（人・場所等）から得られた複数のデータの組。← S03:p.10

⚠️ 比較できるデータ・できないデータ
組み合わせ比較できる？理由
Aさんの年齢 ↔ Aさんのプレイ時間○ できる「Aさん」という共通点あり
Aさんの年齢 ↔ Bさんのプレイ時間× できない共通点がなく比較の基準がない
🎤 音声補足 ← S05a
第4回では全て「対応のあるデータ」を前提としてお話をするということを前提条件として覚えておいてください。

1-3. 次元（データサイエンス文脈）

定義次元: データの種類（組）の数。日常語の「2次元＝平面」とは異なる概念。← S03:p.11

例 年齢・職業・プレイ時間・満足度の4種類のデータ → 4次元データ

50人が回答したデータなら「4次元のデータが50個」と表現する。

🎤 音声補足 ← S05a
データサイエンスを勉強していると「多次元」という単語がたくさん出てくる。多次元という単語が出てきたら「複数のデータを比較しているんだな」とすぐに思い浮かべられるようにしましょう。

1-4. クロス集計表（クロス表）

定義クロス集計表: 2種類以上のデータの組み合わせの度数を表形式で表現したもの。質的・量的データ両方に対応可能。← S03:p.14

例として年齢×プレイ時間のクロス集計表を見てみる。← S03:p.14-16

〜9時間10〜20時間21時間〜計
10代2068
20代251017
30代010515
40代1405
50代0235
計5212450

▲ 「20代の行」と「10〜20時間の列」が交差するセルの値5が、「20代でプレイ時間10〜20時間の人数（度数）」を表す。

🎤 音声補足 ← S05a
非常に簡単な表だが、直感的に2つのデータの関係性がうまく表現できている。ただし「この表を一言で説明してください」と言われると難しい。→ 視覚的に見るには散布図が有効。

1-5. 散布図

定義散布図: 量的データ2変数の分布を2次元座標上の点で可視化した図。← S03:p.17

⚠️ 散布図は量的データ専用

質的データ（性別・満足度など）を散布図で表現すると、複数のデータが同一座標に重なり分布が確認できない。自分で散布図を使う際は、表示したいデータが量的データであることをまず確認すること。

🎤 音声補足 ← S05a
気象庁の報告書に掲載されている東京のスーパーマーケットにおける週平均気温とファミリーアイス販売数の散布図が実例として紹介された。赤い点が昇温期（2月〜7月、約100週分）、青い点が降温期（8月〜1月、約82週分）。

1-6. 散布図の読み方① ― 外れ値への注目

外れ値がある場合、計測ミスか特別な状況かを考察することでデータの概要が把握できる。← S03:p.20-21

🎤 音声補足（かまぼこの事例） ← S05a
かまぼこの販売数の散布図に寒い時期の外れ値が6個ある。3年間のデータなので毎年2週間ほど販売数が急上昇する時期がある。→ お正月の需要であることが一瞬で分かる。日本文化を知らない人がこのデータを見たら計測ミスと考えてしまうかもしれない。データ分析の知識だけでなく、そのデータに関する専門知識も重要。

1-7. 散布図の読み方② ― グループ別（層別）の違いへの注目

一見傾向が分からない散布図でも、グループ別（層別）に色分けすると意味が変わることがある。← S03:p.22-24

🎤 音声補足（鍋用つゆの事例） ← S05a
鍋用つゆの販売数を白黒散布図で見ると中央に輪っかのようなデータがあるだけ。しかし昇温期（赤）・降温期（青）に色分けすると、同じ平均気温でも温度が上がっている時期と下がっている時期では売れ方が違うことが一瞬で分かる。何か傾向がありそうな散布図があったら「何かグループ分けができないか」を考えてみることも重要な視点。

第2章　相関係数（1）

2-1. 相関とは

定義相関: 2つのデータ間の関係のこと。特に直線関係に近い関係の場合に「相関関係にある」と表現する。← S03:p.29

🎤 音声補足 ← S05a
一次関数を覚えている人は、直線の傾きの値が正か負かに対応していると考えると非常に理解しやすい。「直線関係に近い関係」を数字で客観的に表現したものが相関係数。

2-2. 相関係数の符号

相関係数は $r_{AB}$ と表記し、符号（正負）と値の2つで読む。第2章では符号のみを扱う。← S03:p.33

相関の種類相関係数の符号
正の相関rAB = 正の値（+）
無相関rAB = 0
負の相関rAB = 負の値（−）

2-3. 偏差の復習と2次元への拡張

定義偏差: 個々のデータが平均からどれくらい離れているか。偏差 = xᵢ − x̄← S03:p.34

2次元座標（散布図）上のデータでも同じ考え方が使える。← S03:p.35-39

📐 計算例：点 (2, 1)、平均 (4, 4) の場合

A方向の偏差: 2 − 4 = −2

B方向の偏差: 1 − 4 = −3

2-4. 共分散

定義共分散: 「(Aの偏差) × (Bの偏差) の和」の平均。2変数のばらつきの方向性を表す。← S03:p.40

🎤 音声補足 ← S05a
1変数の分散が「(偏差)×(偏差)の平均」だったのに対し、2変数では「(Aの偏差)×(Bの偏差)の平均」とする。もともとの「分散」が2つになったから「共分散」と考えれば大して変わった概念ではない。

5つのデータ (2,1), (1,3), (4,5), (6,5), (7,6)、平均 (4,4) での計算例：← S03:p.42-44

点Aの偏差Bの偏差積
(2, 1)−2−36
(1, 3)−3−13
(6, 5)212
(4, 5)010
(7, 6)326
合計 ÷ 5（データ数）3.4（共分散）

2-5. 共分散の符号と相関の関係

🎤 音声補足 ← S05a
共分散がプラスかマイナスかが分かれば、その時点でデータが平均に対してどのエリアに散らばっているかが分かる。→ ただし、この共分散だけではまだ「相関係数」とは言えない。値の求め方は第3章へ。

第3章　相関係数（2）

3-1. 共分散の問題点：単位

共分散には単位が残ってしまう。← S03:p.57

⚠️ 単位が違うと比較できない

データAが身長(cm)、データBが体重(kg) → 共分散の単位は cm × kg

1組がcm計測・2組がm計測 → 単位の違いで共分散の値が変わり、2組間で比較できない。

→ 単位の影響をなくす必要がある。そこで登場するのが相関係数。

3-2. 相関係数の定義と計算

定義相関係数 $r_{AB}$: 共分散を2つのデータの標準偏差で割ったもの。単位が消え、−1から1の範囲に収まる。← S03:p.58

相関係数の式
rAB = (データAとデータBの共分散) / (データAの標準偏差 × データBの標準偏差)
🎤 音声補足（単位が消える理由） ← S05a
分子（共分散）の単位は cm×kg。分母（標準偏差の積）の単位も cm×kg。割り算すると単位が消える。つまりこの式にすることで共分散の符号は残しつつ、単位を削除することもできる便利な式になっている。

計算例（前章の続き）：← S03:p.60-61

共分散3.4
データAの標準偏差 σA≈ 2.28
データBの標準偏差 σB≈ 1.79
相関係数 rAB= 3.4 / (2.28 × 1.79) ≈ 0.83（強い正の相関）
🎤 音声補足 ← S05a
「≈」（波線2つ）は「だいたいこれくらいの数字」という近似記号。小数点以下を丸めた数字であることを示す。

3-3. 相関係数の強さの目安

※分野によって基準は異なる。以下はあくまで一例。← S03:p.62

相関係数の範囲解釈
0.7 < r ≤ 1強い正の相関
0.4 < r ≤ 0.7中程度の正の相関
0.2 < r ≤ 0.4弱い正の相関
−0.2 ≤ r ≤ 0.2無相関
−0.4 ≤ r < −0.2弱い負の相関
−0.7 ≤ r < −0.4中程度の負の相関
−1 ≤ r < −0.7強い負の相関
🎤 音声補足 ← S05a
例えば0.4でも「相関がない」とみなす分野もある。どの程度の相関係数を持って相関があると認めるかは、扱うデータによって異なるので、ここの表は一例として捉えること。

3-4. 相関係数の留意点（3つ）

← S03:p.63-68

① 相関係数を求めたら必ず散布図を書く

理由1: 外れ値の影響を確認するため

実際には無相関でも、外れ値が1点あるだけで平均の位置がずれ、相関係数が一気に大きくなる。共分散を求める際に平均を使うため、相関係数は外れ値に非常に弱い。

理由2: 直線関係でない散布図があるため

相関係数が0（無相関）でも、散布図がWの字・円・X字などのパターンを示す場合がある。そこからグループ別の影響を仮説として立てることができる。

② データ数の不足に気をつける

データが少ない（例：3点のみ）と、たまたまその点が負の相関の位置関係にあるだけで、全体は正の相関というケースがある。十分なデータ数があるか確認すること。

③ 疑似相関に気をつける

定義疑似相関: 第三の変数（交絡変数）の影響で直接関係がないのに相関があるように見える現象。

例 子どもの身長↑ → 語彙数↑ に見えるが、実際は年齢が両方に影響しているだけ。

第4章　データの比較事例

4-1. 実際のデータ分析では計算機を使う

相関係数の手計算は現実のデータ分析では行わない。理由：計算ミスの危険、データ数が膨大。← S03:p.74

ツール関数名
Excel・GoogleスプレッドシートCORREL関数
PythonPandas の corr 関数
Rcor 関数
🎤 音声補足 ← S05a
一般に「相関係数」といえばピアソンの積率相関係数を指すことが多い。他にスピアマンの順位相関係数なども有名だが、基本的には相関係数といえばピアソンを指すと考えてよい。

4-2. 演習概要

対象データ: 都道府県別・年平均気温・最高気温・最低気温（第2回第4章でWeb APIを使って収集）← S03:p.75

使用ツール: Googleスプレッドシート

📋 演習ステップ ← S03:p.77-79
  1. データ整形: A列（調査年）・B列（都道府県）・C列（平均気温）・D列（最高気温）・E列（最低気温）
  2. ピボットテーブルで最高気温（行）× 最低気温（列）を2.5℃間隔でグループ化しクロス集計表を作成
  3. 最高気温・最低気温の2列を選択し散布図を表示
  4. CORREL関数で相関係数を算出

※実際に演習する必要はない。1〜3章の知識が実際のデータ分析でどのように使われるかを観察する。

4-3. 演習結果と考察
